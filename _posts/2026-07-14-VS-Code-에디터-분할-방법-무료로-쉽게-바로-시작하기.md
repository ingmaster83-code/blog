## VS Code 에디터 분할 방법의 기본 이해

### 단축키로 시작하는 빠른 분할

VS Code에서 가장 직관적인 분할 방식은 단축키를 활용하는 것입니다. 현재 열린 파일을 기준으로 우측에 새 창을 열려면 `Ctrl+\`(윈도우/리눅스) 또는 `Cmd+\`(맥)를 누르면 됩니다. 이렇게 하면 두 개의 에디터 그룹이 생겨 동시에 다른 파일을 볼 수 있습니다.

### 상하 분할과 그리드 분할

더 정교한 **코드 에디터 분할**이 필요하다면, 상단 메뉴에서 `보기(View)` → `에디터 레이아웃(Editor Layout)`을 선택하세요. 여기서 상하 분할, 좌우 분할, 또는 2x2 그리드 분할까지 설정할 수 있습니다. 이 기능은 여러 파일을 동시에 참조해야 하는 복잡한 프로젝트에서 특히 유용합니다.
