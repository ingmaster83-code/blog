## vs code 테마 변경 방법 단계별 가이드

### 1단계: WooaVS 접속 및 마음에 드는 테마 고르기

먼저 [WooaVS](https://vskit.wooahouse.com)에 접속합니다. 메인 페이지에는 다양한 무료 vscode 테마가 카테고리별로 정리되어 있습니다. **다크 모드, 라이트 모드, 하이 컨트라스트** 등 자신의 작업 환경에 맞는 테마를 선택할 수 있습니다.

### 2단계: 미리보기로 실수 없이 선택

많은 초보자들이 미리보기 없이 이름만 보고 설치했다가 실제 화면과 달라 당황합니다. WooaVS에서는 각 테마별로 **실제 코드 편집 화면 미리보기**를 제공하므로, 설치 전에 원하는 색감과 가독성을 확인할 수 있습니다. 예를 들어, JavaScript 코드에서 주석 색상이 어떤지, 문자열 강조가 잘 되는지 미리 체크하세요.

### 3단계: 클릭 한 번으로 자동 설치

테마를 선택하면 "VS Code에서 열기" 버튼이 나타납니다. 이 버튼을 클릭하면 자동으로 VS Code가 실행되며 **해당 테마의 설치 페이지**로 이동합니다. 여기서 "설치(Install)" 버튼만 누르면 됩니다.

### 4단계: 적용 키보드 단축키로 바로 변경

설치 후 대부분의 분들이 "어디서 적용하지?"라고 헤맵니다. 이때 **Ctrl+K, Ctrl+T**를 순서대로 누르면 상단에 테마 선택 창이 나타납니다. 방금 설치한 테마 이름을 입력하고 엔터를 누르면 즉시 적용됩니다. 이 단축키를 외워두면 앞으로 테마 변경이 3초면 끝납니다.
