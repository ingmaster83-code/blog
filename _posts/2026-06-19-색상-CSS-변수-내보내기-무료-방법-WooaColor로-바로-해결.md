### 왜 CSS 변수에 색상을 담아야 할까

CSS 변수(커스텀 속성)는 디자인 토큰의 핵심입니다. 한 번 정의해두면 `var(--primary)` 같은 식으로 프로젝트 전체에서 일관되게 참조할 수 있어 유지보수 비용이 급감합니다. 특히 다크 모드나 테마 전환이 필요한 서비스라면, CSS 변수로 컬러 시스템을 구축하는 것이 표준 관행입니다.

### 주요 기능 살펴보기

#### 1. 직관적인 색상 입력 및 편집

WooaColor는 컬러 피커와 직접 HEX 입력을 모두 지원합니다. 기존 팔레트가 있다면 색상 코드를 하나씩 추가하거나, 임의로 색상을 선택해 실시간 미리보기를 확인할 수 있습니다.

#### 2. 자동 CSS 변수 코드 생성

이 도구의 가장 큰 장점은 가벼움입니다. 별도 프로그램 설치가 필요 없으며, 모든 연산은 사용자 브라우저에서만 처리됩니다. 보안에 민감한 프로젝트라도 안심하고 사용할 수 있습니다.
