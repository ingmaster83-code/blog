## WooaColor 사용법 꿀팁 단계별 가이드

### 1. 이미지 업로드 및 색상 추출

**도구 접속하기**
먼저 위 링크를 통해 WooaColor 웹사이트에 접속합니다. 메인 화면 중앙에 있는 드래그 앤 드롭 영역이나 파일 선택 버튼이 보입니다.

**이미지 업로드하기**
원하는 이미지를 해당 영역에 끌어다 놓거나, 컴퓨터에서 직접 선택합니다. JPG, PNG, WEBP 등 주요 이미지 포맷을 지원하며, 최대 10MB까지 업로드 가능합니다.

**자동 색상 추출 확인**
업로드가 완료되면 즉시 이미지에서 주요 색상 6가지가 자동으로 추출되어 팔레트 형태로 표시됩니다. 각 색상 위에 마우스를 올리면 해당 색상의 HEX 코드가 바로 보입니다.

### 2. 색상 상세 정보 확인 및 복사

**색상 코드 확인하기**
팔레트에서 원하는 색상을 클릭하면 상세 패널이 열립니다. 여기서 HEX, RGB, HSL 세 가지 형식의 코드를 한 번에 확인할 수 있습니다.

**원클릭 복사**
각 코드 옆에 있는 복사 버튼을 클릭하면 클립보드에 자동 저장됩니다. 피그마, CSS 파일, 구글 슬라이드 등 원하는 곳에 바로 붙여넣기 하면 됩니다.

**색상 명도와 채도 조정**
일부 색상은 밝기나 채도가 마음에 들지 않을 수 있습니다. 이때는 팔레트 아래 있는 슬라이더를 움직여 명도와 채도를 실시간으로 조정한 후 수정된 색상을 복사하면 됩니다.

### 3. 팔레트 저장 및 내보내기

**팔레트 저장하기**
브라우저를 닫아도 작업 내용을 유지하고 싶다면 오른쪽 상단의 저장 버튼을 누릅니다. 로컬 스토리지에 저장되어 다음에 접속할 때 자동으로 불러옵니다.

**다양한 형식으로 내보내기**
왼쪽 하단의 내보내기 아이콘을 클릭합니다.
- PNG 이미지: 팔레트를 이미지 파일로 저장하여 포트폴리오나 문서에 삽입
- CSS 코드: 웹 개발 시 바로 사용 가능한 CSS 변수 형태로 복사
- SVG 코드: 벡터 그래픽 편집이 필요한 경우 활용

**팔레트 공유하기**
팀원과 협업 중이라면 URL 공유 기능을 사용하세요. 현재 팔레트 상태가 담긴 고유 링크가 생성됩니다. 상대방이 링크만 열면 동일한 팔레트를 볼 수 있어 브랜드 컬러 가이드 공유에 유용합니다.
