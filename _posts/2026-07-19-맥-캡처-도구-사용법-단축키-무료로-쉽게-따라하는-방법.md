맥북을 처음 사용하면 화면 캡처 한 번 하는 것도 낯설게 느껴집니다. 윈도우에서 쓰던 Print Screen 키가 없어서 당황한 경험, 다들 한 번쯤 있으실 겁니다. 오늘은 **맥 캡처 도구 사용법 단축키**를 완전 정복해서, 누구나 30초 만에 원하는 화면을 저장할 수 있도록 도와드리겠습니다.

## 맥 캡처 도구 사용법 단축키 완벽 가이드

맥에는 별도 설치 없이 기본으로 내장된 강력한 캡처 도구가 있습니다. 키보드 단축키만 외우면 작업 효율이 크게 올라갑니다. 초보자도 바로 따라할 수 있도록 핵심 단축키를 정리했습니다.

### 1. 전체 화면 캡처: Command + Shift + 3
가장 기본적인 전체 화면 캡처입니다. 모니터에 보이는 모든 내용을 한 번에 PNG 파일로 저장합니다. 바탕화면에 자동 저장되며, 파일명은 "스크린샷 날짜 시간" 형식입니다.

### 2. 선택 영역 캡처: Command + Shift + 4
마우스 드래그로 원하는 부분만 자를 수 있습니다. 십자선 커서가 나타나면 클릭 후 드래그하여 영역을 지정하세요. 선택한 부분만 깔끔하게 저장됩니다. 영역 선택 중 Esc 키를 누르면 취소할 수 있습니다.

### 3. 창 캡처: Command + Shift + 4 + Space
특정 창(윈도우)만 캡처하고 싶다면 이 단축키를 사용하세요. 카메라 아이콘이 나타나면 원하는 창을 클릭하면 됩니다. 그림자 효과도 함께 저장되어 프레젠테이션 자료에 유용합니다.

### 4. 캡처 도구 실행: Command + Shift + 5
이 단축키를 누르면 하단에 캡처 도구 메뉴가 나타납니다. 화면 녹화, 전체/선택/창 캡처를 선택할 수 있고, 저장 위치와 타이머도 설정 가능합니다. 영상 녹화가 필요할 때 특히 유용한 맥 화면 캡처 방법입니다.

### 5. 클립보드에 저장하기: 단축키 + Control
캡처한 이미지를 파일로 저장하지 않고 바로 클립보드에 복사하고 싶다면, 위 단축키에 Control 키를 추가하세요. 예: Command + Control + Shift + 4. 문서나 메신저에 바로 붙여넣기 할 때 효율적입니다.

## 주의사항 및 팁

- **저장 위치 변경**: Command + Shift + 5를 누른 후 "옵션"에서 저장 위치를 바탕화면, 문서, 클립보드 등으로 변경할 수 있습니다.
- **파일 형식**: 기본 저장 형식은 PNG입니다. 터미널에서 명령어를 수정하면 JPG, PDF 등으로 변경 가능합니다.
- **화면 녹화 시 음성**: 마이크 아이콘을 클릭하면 시스템 사운드나 마이크를 선택해 녹화에 포함할 수 있습니다.
- **단축키 충돌**: 일부 앱에서 같은 단축키를 사용할 수 있으니, 충돌 시 시스템 환경설정 > 키보드 > 단축키에서 확인하세요.
