### 기능 상세: 핵심 기능 3가지

#### 1. 실시간 Debounce 테스트
입력 필드에 텍스트를 입력할 때마다 이벤트가 발생하는 상황을 시뮬레이션합니다. 딜레이 시간(ms)을 조절하면 마지막 입력 후 몇 초 뒤에 함수가 실행되는지 바로 확인할 수 있습니다. 검색어 자동완성, 입력 폼 저장 등에 유용합니다.

#### 2. 실시간 Throttle 테스트
스크롤이나 마우스 움직임처럼 연속적으로 발생하는 이벤트를 시뮬레이션합니다. 간격 시간을 설정하면 일정 시간 동안 한 번만 실행되는 것을 시각적으로 보여줍니다. 무한 스크롤, 애니메이션 프레임 제어에 최적화되어 있습니다.

#### 3. 생성된 코드 내보내기
테스트가 끝나면 설정한 delay와 leading/trailing 옵션까지 포함된 완성된 JavaScript 코드를 즉시 복사할 수 있습니다. 별도의 라이브러리 설치 없이 바로 프로젝트에 붙여넣기만 하면 됩니다.
