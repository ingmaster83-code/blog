### 사용법 단계별 가이드

이 도구는 직관적인 UI 덕분에 누구나 쉽게 사용할 수 있습니다. 아래 단계를 따라 해보세요.

#### 1. 색상 선택하기
사이트에 접속하면 두 개의 기본 색상 선택기가 나타납니다. 첫 번째는 시작 색상, 두 번째는 끝 색상입니다. 원하는 색상을 클릭하거나 HEX 코드를 직접 입력하세요. 중간에 색상을 더 추가하고 싶다면 "색상 추가" 버튼을 누르면 됩니다.

#### 2. 방향과 스타일 설정하기
**CSS 그라데이션 무료** 도구의 핵심은 방향 설정입니다. "각도" 슬라이더를 움직이면 0도부터 360도까지 자유롭게 조정할 수 있습니다. 예를 들어 90도는 왼쪽에서 오른쪽, 180도는 위에서 아래 방향입니다. 또한 "원형(radial)" 모드로 전환하면 동그란 모양의 그라데이션도 만들 수 있습니다.

#### 3. 코드 복사 및 적용
모든 설정이 끝나면 우측의 코드 창에 실시간으로 CSS 코드가 생성됩니다. "복사" 버튼을 누르면 클립보드에 저장됩니다. 이 코드를 여러분의 스타일시트에 붙여넣기만 하면 끝입니다. 이렇게 간단하게 **웹 디자인 그라데이션**을 완성할 수 있습니다.

## 주의사항: 자주 하는 3가지 실수와 해결법

많은 분들이 **CSS 그라데이션 생성 온라인** 도구를 사용할 때도 실수를 합니다. 아래 내용을 꼭 확인하세요.

1. **브라우저 호환성 확인**: 생성된 코드는 `-webkit-`이나 `-moz-` 같은 접두사가 없는 표준 CSS입니다. 오래된 브라우저를 지원해야 한다면 직접 접두사를 추가하세요. WooaDev는 표준 코드를 생성하므로 최신 브라우저 환경에 최적화되어 있습니다.

2. **과도한 색상 포인트 피하기**: 2~3개의 색상이 가장 자연스럽습니다. 5개 이상 사용하면 오히려 어색해질 수 있습니다. **그라데이션 코드 만들기**에서는 심플함이 가장 중요합니다.

3. **명도 대비 확인**: 배경에 텍스트를 올릴 계획이라면 그라데이션의 명도가 너무 낮거나 높지 않은지 반드시 확인하세요. 도구의 실시간 미리보기 기능을 활용하면 이 문제를 예방할 수 있습니다.
