## 기능 상세: WooaImage로 사진 해상도 높이는 방법

### 주요 기능 3가지

1. **AI 업스케일링**  
   저해상도 이미지를 입력하면 AI가 노이즈를 제거하고 디테일을 살려 고해상도 이미지로 변환합니다. 특히 이미지 해상도가 낮은 오래된 사진에서 효과가 두드러졌습니다.

2. **실시간 미리보기**  
   업스케일 전후 결과를 바로 비교할 수 있어, **사진 보정** 작업을 하면서 원하는 수준에 도달했는지 즉시 확인 가능합니다.

3. **브라우저 내 처리**  
   앞서 말씀드렸듯 모든 연산이 사용자의 브라우저 안에서 실행됩니다. 파일 업로드 없이 **무료 사진 편집**이 가능하다는 점에서 프라이버시 중심 사용자에게 안성맞춤입니다.

### 사용 방법 (3단계)

1. WooaImage 웹사이트에 접속합니다.
2. 화질을 개선할 사진 파일을 드래그하거나 클릭하여 업로드합니다.
3. 개선 완료 버튼을 누르고 결과를 확인한 후 다운로드합니다.

복잡한 설정이나 튜토리얼이 필요 없습니다. **이미지 확대** 작업이 처음이라도 직관적인 UI 덕분에 누구나 바로 사용할 수 있습니다.
