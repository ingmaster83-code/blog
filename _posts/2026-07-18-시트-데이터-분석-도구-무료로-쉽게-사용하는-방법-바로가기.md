## 주요 기능: 데이터 분석을 위한 3가지 핵심 기능

WooaSheet는 단순히 데이터를 보여주는 것을 넘어, 실제 업무에 바로 활용할 수 있는 강력한 기능들을 제공합니다.

### 1. CSV 파일 업로드 및 붙여넣기
엑셀에서 내보낸 CSV 파일을 직접 업로드하거나, 표 데이터를 복사해서 텍스트 영역에 붙여넣기만 하면 됩니다. 파일 형식에 구애받지 않고 자유롭게 데이터 분석을 시작할 수 있습니다.

### 2. 자동 데이터 타입 감지 및 요약
숫자, 날짜, 텍스트 등 각 열의 데이터 타입을 자동으로 분석합니다. 숫자 데이터가 있는 열은 자동으로 합계와 평균이 계산되며, 텍스트 데이터는 고유값 개수와 빈도수를 보여줍니다.

### 3. 그룹별 집계와 분포 분석
특정 열을 기준으로 데이터를 그룹화하고, 각 그룹의 통계를 확인할 수 있습니다. 예를 들어 '지역'별로 '매출'의 합계를 보거나, '부서'별 '인원수'의 평균을 쉽게 구할 수 있습니다. 또한 각 열의 데이터 분포를 분석하여 업무에 필요한 인사이트를 빠르게 찾아낼 수 있습니다.

## 활용 팁: 초보자를 위한 단계별 가이드

WooaSheet를 처음 사용하는 분들을 위해 실제 데이터 분석을 어떻게 진행하는지 단계별로 설명해 드립니다.

### 1단계: 데이터 준비하기
분석하고 싶은 데이터를 CSV 형식으로 준비합니다. 엑셀에서 '다른 이름으로 저장'을 선택한 후 'CSV UTF-8' 형식으로 저장하면 가장 호환성이 좋습니다.

### 2단계: 데이터 불러오기
WooaSheet 웹사이트에 접속하여 '파일 선택' 버튼을 클릭하거나, 데이터를 직접 텍스트 영역에 붙여넣기 합니다.

### 3단계: 결과 확인하기
데이터가 업로드되면 자동으로 분석이 시작됩니다. 각 열의 데이터 타입과 함께 합계, 평균, 최대값, 최소값 등의 통계가 표시됩니다. 특정 열의 데이터 분포를 확인하고 싶다면 해당 열의 '분석 보기' 버튼을 클릭하기만 하면 됩니다.

이처럼 복잡한 **데이터 정리** 과정 없이도, 단 3단계만으로 원하는 정보를 얻을 수 있습니다. 회사에서 매일 받는 판매 데이터, 학교에서 사용하는 성적 데이터, 마케팅 분석 자료 등 다양한 분야에서 유용하게 활용할 수 있습니다.
