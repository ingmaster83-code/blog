## FontKit 소개: 행정안전부 무료 폰트를 웹에서 즉시 사용하기

FontKit([FontKit 무료로 사용하기](https://fontkit.wooahouse.com))은 별도의 설치 없이 브라우저에서 바로 공공기관 무료 폰트를 찾고, 적용 코드를 생성해주는 온라인 서비스입니다. 특히 **행정안전부 무료 폰트**인 한국재난안전폰트를 포함한 다양한 공공 폰트를 지원해, 누구나 쉽게 웹사이트와 문서에 활용할 수 있도록 도와줍니다.

## 행정안전부 무료 폰트 사용법: FontKit 기능 상세

FontKit은 단순한 폰트 검색 이상의 기능을 제공합니다. 실제로 어떻게 사용하는지 단계별로 알아보겠습니다.

### 1. 원하는 행정안전부 폰트 검색하기
사이트에 접속하면 검색창이 보입니다. 여기에 "한국재난안전폰트" 또는 "행정안전부"라고 입력하면 관련 폰트가 바로 나타납니다. 공공기관 무료 폰트뿐 아니라 나눔고딕, 나눔명조, 본고딕 등 국내 인기 무료 폰트도 모두 검색됩니다.

### 2. 폰트 미리보기와 스타일 확인
검색된 폰트를 클릭하면 다양한 굵기(Thin, Regular, Bold)와 스타일을 실시간으로 미리볼 수 있습니다. **무료 폰트 다운로드** 없이도 내 콘텐츠에 어떤 느낌일지 바로 확인할 수 있어 편리합니다.

### 3. CSS 코드 자동 생성
원하는 폰트와 스타일을 선택하면, FontKit이 자동으로 **웹 폰트 적용**에 필요한 CSS @import 코드와 링크 태그를 생성해줍니다. 이 코드를 복사해서 웹사이트의 `<head>` 부분이나 CSS 파일에 붙여넣기만 하면 됩니다.
