## CSV SQL 변환 무료 온라인 도구 WooaSheet 첫인상

처음 [WooaSheet](https://wooasheet.wooahouse.com)에 접속했을 때 가장 놀란 점은 '심플함'이다. 광고 하나 없이 깔끔한 입력창 하나만 덩그러니 있다. 로그인도, 회원가입도 필요 없다. 내가 가진 CSV 데이터를 복사해서 붙여넣기만 하면 끝이다. SQL에 대한 사전 지식이 거의 없는 나도 10초 만에 사용법을 익혔다.

이 도구는 CSV 데이터를 분석해서 자동으로 테이블명과 컬럼명을 추출하고, 그에 맞는 완전한 SQL INSERT문을 생성해준다. 특히 첫 행을 컬럼명으로 인식하는 기능이 똑똑해서, 헤더가 있는 CSV라면 별도 설정 없이 바로 변환된다.

## 기능 상세: INSERT문 자동 생성과 데이터 타입 추론

### CSV 붙여넣기와 자동 파싱

WooaSheet의 핵심 기능은 단순하다. 텍스트 영역에 CSV 데이터를 붙여넣고 '변환' 버튼을 누르면 된다. 숫자, 문자열, 날짜 등 데이터 타입을 자동으로 추론해서 각 컬럼에 맞는 SQL 타입을 지정해준다. 예를 들어 '2024-01-15' 같은 값은 DATE로, 숫자만 있는 칼럼은 INT나 DECIMAL로 자동 설정된다.

### SQL 변환 옵션 커스터마이징

고급 사용자를 위해 몇 가지 옵션도 준비되어 있다. 테이블명을 직접 지정할 수 있고, INSERT IGNORE나 REPLACE 같은 키워드를 선택할 수 있다. 대용량 데이터를 처리할 때는 100개 행씩 묶어서 배치 INSERT문을 생성하는 기능도 유용하다.

## 실제 사용 후기: 엑셀 데이터를 DB로 옮긴 경험

어제 실제 업무에서 사용해봤다. 고객 주문 내역이 담긴 CSV 파일을 열어서 내용을 전체 복사(Ctrl+A, Ctrl+C)한 뒤, WooaSheet 페이지에 붙여넣었다. 약 500행짜리 데이터였는데, 변환 버튼을 누르자 1초도 안 돼서 완성된 SQL문이 출력됐다.

생성된 쿼리를 MySQL Workbench에 바로 붙여넣고 실행했더니 오류 없이 모든 데이터가 정상적으로 INSERT되었다. 직접 손으로 썼다면 오타나 따옴표 처리 실수로 최소 3~4번은 디버깅했을 텐데, 이 도구는 구문 오류가 전혀 없었다. 특히 문자열 안에 작은따옴표(')가 포함된 주소 데이터도 자동으로 이스케이프 처리해줘서 안심이 됐다.

이 도구의 가장 큰 장점은 '무료'라는 점이다. 다른 SQL 변환기들은 건당 제한이 있거나 워터마크를 붙이는 경우가 많지만, WooaSheet는 횟수 제한 없이 무료로 사용할 수 있다. 게다가 데이터가 서버로 전송되지 않아서 보안에 민감한 업무 데이터도 안심하고 사용할 수 있다.
