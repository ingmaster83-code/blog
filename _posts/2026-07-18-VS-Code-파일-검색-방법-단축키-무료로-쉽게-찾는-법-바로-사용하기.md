## VS Code 파일 검색 방법 단축키로 속도 높이기

VS Code에서 파일을 빠르게 찾는 가장 기본적인 방법은 단축키를 익히는 것입니다. 하지만 많은 분들이 이 기능을 제대로 쓰지 못해 불편을 겪습니다. 예를 들어, 파일 이름을 정확히 몰라서 검색 결과가 안 나오거나, 여러 탭이 열려 있어 원하는 파일을 놓치는 경우가 대표적입니다. 이런 문제를 해결하려면 **vs code 파일 검색 방법**을 체계적으로 배우고, 보조 도구를 활용하는 것이 핵심입니다.

### 기본 단축키와 한계 인식하기

VS Code의 기본 파일 검색 단축키는 `Ctrl+P`(윈도우) 또는 `Cmd+P`(맥)입니다. 이 키를 누르면 입력창이 뜨고, 파일 이름 일부만 입력해도 실시간으로 결과가 나타납니다. 하지만 여러 프로젝트를 동시에 다루거나, 파일 개수가 수백 개를 넘어가면 이 방법만으로는 한계가 있습니다. 저도 처음에는 단축키만 믿다가 검색에 실패한 경험이 많았습니다.

### 연관 검색어 활용의 중요성

검색 효율을 높이려면 연관 검색어(LSI)를 자연스럽게 활용해야 합니다. 예를 들어, "설정 파일"을 찾을 때는 `settings.json`처럼 정확한 이름을 몰라도 `config`, `env`, `option` 같은 단어를 함께 사용하면 더 빠르게 접근할 수 있습니다. 이처럼 **파일 찾기 팁**을 알면 단순한 단축키 이상의 효과를 볼 수 있습니다.
