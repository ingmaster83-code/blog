## 주의사항 및 팁

- **파일 크기**: 50MB 이상의 대용량 XML은 브라우저 성능에 따라 지연이 발생할 수 있습니다. 필요한 경우 파일을 분할하여 변환하세요.
- **인코딩**: UTF-8 인코딩을 권장합니다. 다른 인코딩(euc-kr 등)은 변환 전에 변환기를 통해 미리 확인하세요.
- **특수 문자**: XML에 특수 문자가 포함된 경우 JSON 이스케이프 처리가 자동으로 적용됩니다. **JSON 변환기**의 출력이 깨진다면 입력 XML의 문법을 먼저 점검하세요.
- **구조 확인**: JSON은 중첩 구조를 지원하지만, XML의 속성(attribute)은 JSON에서 별도 키로 매핑됩니다. **XML 파싱** 결과를 검토하여 데이터 무결성을 확인하세요.
