## 온라인 코드 실행시 자주 하는 3가지 실수와 해결법

### 1. 코드 저장 없이 새로고침하는 실수

많은 초보자분들이 **온라인 IDE**에서 코드를 작성하다가 실수로 페이지를 새로고침하면 모든 작업이 날아가는 경험을 합니다. 브라우저 기반 도구는 로컬 저장소를 활용하기 때문에, 작업 중간에 자동 저장 기능을 믿지 말고 주기적으로 코드를 복사해두는 습관이 필요합니다.

### 2. 콘솔 오류 무시하기

코드가 실행은 되는데 원하는 결과가 나오지 않을 때, 대부분의 사람들은 코드만 다시 씁니다. 하지만 **코드 테스트 도구**의 핵심은 콘솔 패널을 활용하는 것입니다. WooaDev에서는 우측 하단에 실시간 오류 메시지가 표시되므로, 빨간색 오류를 클릭하면 정확한 줄 번호와 원인을 알 수 있습니다.

### 3. 브라우저 호환성 간과하기

특정 브라우저에서만 잘 돌아가는 코드를 작성하는 경우가 많습니다. **무료 코딩 도구**를 사용할 때는 크롬, 파이어폭스, 엣지 등 여러 브라우저에서 테스트해보는 것이 좋습니다. WooaDev는 크롬 기반으로 최적화되어 있지만, 표준 웹 API를 사용하면 대부분의 브라우저에서 동일하게 작동합니다.

### 2단계: 언어 선택 및 코드 작성
화면 좌측 상단에서 JavaScript, HTML, CSS 중 원하는 언어를 선택합니다. 초보자라면 JavaScript를 추천드리며, 간단한 Hello World부터 시작해보세요.
