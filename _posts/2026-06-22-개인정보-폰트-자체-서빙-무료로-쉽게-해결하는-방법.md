며칠 전, 클라이언트 웹사이트에서 폰트 라이선스 문제로 골머리를 앓았습니다. 구글 폰트를 쓰자니 사이트 속도가 느려지고, 직접 서빙하려니 비용이 부담됐죠. 그러다 우연히 발견한 도구 덕분에 **개인정보 폰트 자체 서빙 무료**로 해결할 수 있었습니다.

## FontKit: 개인정보 걱정 없는 무료 폰트 자체 서빙 도구

[FontKit 무료로 사용하기](https://fontkit.wooahouse.com)는 브라우저 기반으로 작동하는 웹폰트 변환 도구입니다. 설치나 회원가입이 전혀 필요 없고, 모든 파일 처리가 사용자 로컬에서만 이루어집니다. 제가 직접 사용해본 결과, 개인정보 유출 위험 없이 안전하게 폰트를 자체 서빙할 수 있었습니다.

### 주요 기능 상세

#### 1. 온라인 폰트 변환과 서브셋
TTF나 OTF 파일을 업로드하면 자동으로 WOFF2로 변환해줍니다. 특히 서브셋 기능이 인상적이었는데, 웹사이트에서 실제로 사용하는 문자만 추출해 폰트 용량을 획기적으로 줄일 수 있습니다. 제가 테스트한 폰트는 원래 5MB였는데, 서브셋 적용 후 200KB로 줄었습니다.

#### 2. 웹폰트 서빙 최적화
변환된 폰트는 바로 CSS 코드와 함께 제공됩니다. @font-face 규칙을 복사해서 붙여넣기만 하면 끝입니다. 서버에 직접 파일을 올리고 코드만 수정하면 되니, **웹폰트 최적화** 작업이 정말 간단해집니다.

#### 3. 개인정보 보호
이 도구의 가장 큰 장점은 개인정보 보호입니다. 모든 변환 과정이 브라우저 안에서 이루어지기 때문에 파일이 외부 서버로 전송되지 않습니다. 제가 사용한 유료 폰트 파일도 안심하고 업로드할 수 있었습니다.

### 실제 사용 후기

처음에는 '설마 이렇게 쉽게 될까?' 싶었습니다. 하지만 실제로 해보니 정말 간단했습니다.

1. FontKit 사이트에 접속
2. TTF 폰트 파일 업로드
3. 서브셋 옵션에서 필요한 문자만 선택
4. 변환 버튼 클릭

30초도 걸리지 않았습니다. 변환된 WOFF2 파일과 CSS 코드가 바로 다운로드됐고, 제 웹사이트에 적용하니 폰트 로딩 속도가 눈에 띄게 빨라졌습니다. 특히 **무료 폰트 서빙**이 가능하다는 점이 가장 만족스러웠습니다.
