## 배민 무료 폰트 사용법, 왜 계속 실패할까?

디자인 작업을 하다 보면 배민의 감성적인 폰트를 꼭 써보고 싶은 순간이 찾아옵니다. 하지만 막상 **배민 무료 폰트 사용법 방법**을 검색해 보면 "CSS에 @font-face를 추가하세요"라는 복잡한 설명만 가득합니다. 폰트 파일을 직접 다운로드하고, 포맷을 변환하고, 경로를 수정하다 보면 어느새 시간만 한 시간째 날아가 있습니다. 특히 윈도우 유저라면 TTF 파일을 웹용으로 변환해야 하는 번거로움 때문에 포기한 경험이 한 번쯤 있을 겁니다. 이 글에서는 이런 실수를 완전히 차단하고, **무료 온라인 병합 도구**인 FontKit 하나로 바로 해결하는 방법을 알려드립니다.

## 해결책: FontKit으로 30초 만에 끝내기

위 문제의 핵심은 **브라우저 환경과 폰트 파일 포맷의 불일치**에 있습니다. 대부분의 사람이 배민 폰트를 다운로드만 하고, 웹에서 쓸 수 있는 woff2 포맷으로 변환하지 않기 때문에 오류가 발생합니다. [FontKit 무료로 사용하기](https://fontkit.wooahouse.com)는 이 문제를 정확히 짚어주는 도구입니다. 설치도, 회원가입도 필요 없이 브라우저 하나만 있으면 됩니다. 이 도구는 단순히 폰트를 변환하는 것을 넘어서, **폰트 추출**과 **폰트 미리보기** 기능까지 제공하므로 초보자도 쉽게 접근할 수 있습니다.

## 배민 무료 폰트 사용법: 단계별 가이드

### 1단계: 배민 폰트 파일 준비하기

먼저 배민의 공식 사이트에서 원하는 폰트를 다운로드합니다. 보통 OTF나 TTF 형식으로 제공됩니다. 이 단계에서 자주 하는 실수는 "파일 이름에 한글이나 띄어쓰기를 그대로 두는 것"입니다. FontKit으로 가져올 때는 문제가 없지만, 나중에 CSS 경로를 잡을 때 오류의 원인이 됩니다. 파일 이름을 "bmfont.ttf"처럼 영문과 소문자로 미리 바꿔두는 습관을 들이세요.

### 2단계: FontKit에 업로드하기

[FontKit](https://fontkit.wooahouse.com) 사이트에 접속하면 중앙에 큰 업로드 버튼이 보입니다. 여기에 준비한 배민 폰트 파일을 드래그 앤 드롭으로 넣으면 됩니다. 이때 주의할 점은 한 번에 여러 폰트를 올려도 된다는 것입니다. 예를 들어 '배민 한나체'와 '배민 도현체'를 동시에 올리면 한 번에 변환할 수 있어 효율적입니다. 파일이 업로드되면 자동으로 **브라우저 폰트** 호환 포맷인 woff2로 변환됩니다.

### 3단계: CSS 코드와 파일 다운로드 받기

변환이 완료되면 화면에 두 가지 결과가 나타납니다. 첫째는 바로 쓸 수 있는 CSS 코드입니다. 이 코드를 복사해서 웹사이트의 `<head>` 태그 안에 붙여넣기만 하면 됩니다. 둘째는 변환된 woff2 파일 다운로드 링크입니다. 이 파일을 서버에 업로드한 후 CSS 경로만 수정하면 끝입니다. **무료 웹 폰트 생성기**의 장점은 이 모든 과정이 30초 안에 끝난다는 점입니다.

### 4단계: 실제 웹사이트에 적용하기

CSS 코드를 적용한 후 브라우저에서 사이트를 새로고침 해보세요. 만약 폰트가 여전히 적용되지 않는다면, 대부분의 원인은 **캐시 문제**입니다. 브라우저의 개발자 도구(F12)를 열어 Network 탭에서 폰트 파일이 404 오류를 내는지 확인하세요. 경로가 잘못되었다면 상대 경로 대신 절대 경로를 사용하는 것이 가장 확실한 해결법입니다.

## 주의사항: 자주 하는 실수 TOP 3

첫째, **폰트 라이선스를 반드시 확인하세요.** 배민 폰트는 개인 및 상업용으로 무료이지만, 재배포나 수정에는 제한이 있을 수 있습니다. FontKit은 변환만 해줄 뿐 원본 파일의 라이선스를 변경하지 않습니다.

둘째, **모든 브라우저에서 테스트하세요.** FontKit이 woff2로 변환해 주지만, 구형 브라우저(IE 11 등)에서는 지원하지 않을 수 있습니다. 이런 경우 CSS에 fallback 폰트를 지정해 두는 것이 안전합니다. 예를 들어 `font-family: 'BM Hanna', 'Malgun Gothic', sans-serif;`처럼 말이죠.

셋째, **과도한 폰트 파일 개수는 사이트 속도를 늦춥니다.** 배민 폰트의 여러 스타일(Regular, Bold 등)을 한꺼번에 적용하면 로딩 시간이 길어집니다. 꼭 필요한 굵기만 선택해서 **온라인 폰트 경량화**를 고려하세요.
