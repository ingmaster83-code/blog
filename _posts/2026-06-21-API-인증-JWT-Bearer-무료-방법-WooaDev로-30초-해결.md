## 기능 상세: 어떻게 30초 만에 API 인증을 끝내나?

### JWT 생성에 필요한 모든 요소를 자동 구성

WooaDev의 JWT Bearer 도구는 개발자가 가장 자주 사용하는 인증 토큰 생성 시나리오를 완벽하게 지원합니다. 기본적으로 `HS256` 알고리즘을 사용하며, 사용자 ID나 이메일 같은 `sub(Subject)` 클레임과 `iat(Issued At)`와 `exp(Expiration Time)` 같은 시간 관련 클레임을 자동 생성합니다.

토큰 생성의 핵심인 시크릿 키(Secret Key)를 안전하게 입력할 수 있는 전용 필드가 마련되어 있습니다. "내 시크릿 키를 어디에 저장하지?"라는 고민을 덜어주며, 브라우저 로컬에만 임시 저장되므로 보안 걱정도 없습니다.

### 실시간 토큰 미리보기

설정을 변경할 때마다 생성된 JWT 토큰이 실시간으로 미리보기 됩니다. `Header.Payload.Signature` 구조로 이루어진 Bearer 토큰을 바로 복사해서 API 테스트에 활용할 수 있습니다.
