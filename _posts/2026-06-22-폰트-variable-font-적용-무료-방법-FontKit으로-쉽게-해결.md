# Variable font 적용이 막막하다면? FontKit 하나로 끝

웹사이트 속도를 높이기 위해 폰트 최적화를 고민하던 중, **variable font(가변 폰트)** 라는 개념을 접한 적이 있으신가요? 하나의 폰트 파일로 굵기, 폭, 기울기 등 모든 스타일을 표현할 수 있어 웹 성능에 혁신을 가져다주지만, 막상 적용하려면 복잡한 코드와 전문 지식이 필요해 좌절한 경험이 많을 것입니다. 특히 기존 폰트를 가변 폰트로 변환하거나, 원하는 축(axis)만 추출하는 작업은 개발자가 아니면 엄두도 내기 어렵습니다.

## Variable font 적용이 어려운 진짜 이유

기존의 일반 폰트(static font)는 각 굵기(Regular, Bold, Light 등)마다 별도의 파일을 불러와야 했습니다. 이는 HTTP 요청 수를 증가시키고 전체 페이지 로딩 속도를 저하시키는 주범입니다. 반면 **variable font(가변 폰트)** 는 하나의 파일로 무한에 가까운 스타일 변형을 지원하여 로딩 속도를 획기적으로 개선합니다.

문제는 가변 폰트를 직접 만들거나, 기존 폰트에서 불필요한 축(axis)을 제거하는 과정이 매우 까다롭다는 점입니다. 대부분의 온라인 도구는 기능이 제한적이거나 유료이며, 전문 소프트웨어는 설치와 학습에 많은 시간이 소요됩니다. 바로 이 지점에서 **무료 온라인 variable font 변환 도구**의 필요성이 절실해집니다.

## FontKit: 무료로 variable font를 쉽게 적용하는 최고의 도구

이 모든 고민을 해결해 줄 도구가 바로 [FontKit](https://fontkit.wooahouse.com)입니다. **FontKit**은 브라우저 기반의 무료 폰트 최적화 도구로, 복잡한 설치나 회원가입 없이 누구나 쉽게 **variable font를 적용**할 수 있습니다. 특히 **구글 폰트 최적화**나 **웹폰트 용량 줄이기** 작업이 필요한 웹 퍼블리셔, 개발자, 블로거에게 필수적인 서비스입니다.

### 주요 기능: variable font 변환과 최적화를 한 번에

FontKit이 제공하는 핵심 기능은 다음과 같습니다.

#### 1. 기존 폰트를 variable font로 변환

일반 TTF 또는 OTF 폰트 파일을 업로드하면 자동으로 가변 폰트로 변환해 줍니다. 변환 과정에서 불필요한 축(Axis)을 선택적으로 제거하여 파일 크기를 더욱 줄일 수 있습니다. 예를 들어, 'wght'(굵기) 축만 필요하다면 나머지 축은 삭제하여 **폰트 최적화** 효율을 극대화할 수 있습니다.

#### 2. 폰트 서브셋(Subset) 기능

Latin, Cyrillic, Hangul 등 필요한 언어 문자만 추출하여 폰트 용량을 획기적으로 줄입니다. 한국어 블로그라면 한글, 영문, 숫자만 남기고 나머지 언어 문자를 제거하여 **variable font 적용** 후에도 빠른 로딩 속도를 유지할 수 있습니다.

### 활용 팁: FontKit으로 웹사이트 성능 극대화하기

**1. Google Fonts 커스터마이징**
Google Fonts에서 제공하는 Noto Sans KR, Roboto 등 인기 폰트를 다운로드한 후, FontKit에 업로드하여 굵기 범위를 300~700으로 제한하고 한글만 서브셋팅하세요. 그러면 원본 대비 70% 이상 용량이 줄어든 **가변 폰트**를 얻을 수 있습니다.

**2. CSS @font-face와 연동**
변환된 폰트를 다운로드한 후, CSS에 `font-weight` 값 대신 `font-variation-settings` 속성을 사용하여 적용합니다. 예를 들어 `font-variation-settings: 'wght' 500;`과 같이 선언하면 **variable font 적용**이 완료됩니다. FontKit은 이 과정에 필요한 설정값을 직관적으로 제공하므로, CSS 코드만 복사하여 붙여넣으면 됩니다.
