### JSON 구조를 XML로 변환할 때 주의할 점
JSON의 배열(array)은 XML에서 반복되는 태그로 변환됩니다. 예를 들어 `"items": [1, 2, 3]`은 `<items>1</items><items>2</items><items>3</items>` 형태로 바뀝니다. 변환 전에 데이터 구조를 간단히 정리하면 더 깔끔한 **XML 변환 도구** 결과물을 얻을 수 있습니다.

### XML을 JSON으로 변환할 때 활용법
XML 속성(attribute)은 JSON에서 `@속성명` 형태의 키로 변환됩니다. 이 점을 미리 알고 작업하면 **데이터 형식 변환** 시 혼란을 줄일 수 있습니다. 특히 RSS 피드나 Sitemap XML을 다룰 때 강력한 무료 변환기 역할을 합니다.
