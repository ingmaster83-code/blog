## 사용법 단계별 가이드

### 1단계: 준비물과 접속

먼저 변환하고 싶은 SVG 아이콘 파일을 컴퓨터에 준비합니다. 디자인 툴에서 내보내기 하거나, 무료 아이콘 사이트에서 다운로드 받은 SVG 파일이면 충분합니다. 파일이 준비되면 [WooaDev 아이콘 폰트 생성기](https://wooadev.wooahouse.com)에 접속합니다.

### 2단계: 파일 업로드 및 기본 설정

페이지 중앙에 있는 파일 선택 버튼을 클릭하여 준비한 SVG 파일들을 업로드합니다. 한 번에 여러 개의 아이콘을 업로드할 수 있어서 프로젝트 전체의 아이콘을 한 번에 폰트로 만들 수 있습니다. 업로드가 완료되면 폰트의 기본 이름과 유니코드 매핑을 자동으로 설정해주므로, 따로 수정하지 않아도 바로 다음 단계로 넘어갈 수 있습니다.

### 3단계: CSS 코드 생성 및 다운로드

모든 아이콘이 업로드되었다면 생성 버튼을 클릭합니다. 잠시 기다리면 아이콘 폰트 CSS 코드와 함께 WOFF, WOFF2 형식의 폰트 파일이 자동으로 패키징되어 제공됩니다. 생성된 결과물은 크게 두 가지로 나뉩니다.

- **CSS 코드**: 웹사이트에 복사해서 붙여넣기만 하면 되는 완성된 코드
- **폰트 파일**: 서버에 업로드할 실제 폰트 파일들

이 코드를 HTML 파일의 `<style>` 태그나 별도의 CSS 파일에 삽입한 후, 원하는 위치에 `<i class="icon-name"></i>` 형식으로 태그를 작성하면 아이콘이 즉시 화면에 표시됩니다.
