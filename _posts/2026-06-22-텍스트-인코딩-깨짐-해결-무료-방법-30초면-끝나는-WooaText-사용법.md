## 텍스트 인코딩 깨짐, 왜 매번 당황할까?

한국에서 문서 작업을 하다 보면 **텍스트 인코딩 깨짐 해결**이 급하게 필요한 순간이 찾아옵니다. 이메일로 받은 CSV 파일을 열었는데 한글이 전부 "ㅁㅁㅁ"이나 "¿ë¶?" 같은 알 수 없는 문자로 변해 있거나, 오래된 웹페이지 소스를 복사했더니 글자가 깨져서 하나도 읽히지 않는 경험, 한 번쯤 있으시죠?

이 문제의 원인은 간단합니다. EUC-KR, UTF-8, CP949 등 서로 다른 문자 인코딩 방식이 충돌하기 때문입니다. 특히 한국에서는 EUC-KR(완성형)과 UTF-8(유니코드)이 혼용되다 보니 인코딩 오류가 빈번하게 발생합니다. 하지만 매번 변환 사이트를 찾거나 프로그램을 설치하는 번거로움을 감수하기엔 시간이 아깝습니다.

오늘은 이런 고민을 단 30초 만에 해결해주는 무료 도구, **WooaText**를 소개합니다. 설치도, 회원가입도 필요 없습니다.

## 무료 텍스트 인코딩 깨짐 해결 도구: WooaText

[WooaText 무료로 사용하기](https://textkit.wooahouse.com)

WooaText는 웹 브라우저에서 바로 작동하는 온라인 텍스트 인코딩 변환기입니다. 별도 소프트웨어 설치 없이도 깨진 한글을 원래 상태로 복원해줍니다. 자주 사용하는 인코딩 변환은 물론, **자동 인코딩 감지 기능**까지 지원하여 어떤 깨짐 현상에도 빠르게 대응할 수 있습니다.

- **설치 불필요**: 인터넷만 연결되면 즉시 사용
- **자동 감지**: 인코딩 종류를 몰라도 자동으로 찾아 변환
- **다양한 포맷 지원**: EUC-KR, UTF-8, CP949, ISO-8859-1 등

## 텍스트 인코딩 깨짐 해결 방법: 단계별 사용법

### 1단계: 깨진 텍스트 준비하기
먼저 복원하고 싶은 깨진 텍스트를 준비합니다. CSV 파일, 메모장 문서, HTML 코드 등 어떤 형식이든 상관없습니다. 텍스트만 복사하거나 파일 자체를 업로드하면 됩니다.

### 2단계: WooaText 접속하기
브라우저를 열고 [WooaText 공식 페이지](https://textkit.wooahouse.com)로 이동합니다. 페이지가 열리면 별다른 설명 없이 깔끔한 입력창이 보입니다.

### 3단계: 텍스트 입력 후 인코딩 변환 실행
입력창에 깨진 텍스트를 붙여넣거나 파일을 드래그하여 업로드합니다. 상단 드롭다운 메뉴에서 현재 인코딩(예: EUC-KR)과 변환할 인코딩(예: UTF-8)을 선택하거나, **자동 감지**를 켜두면 도구가 알아서 인코딩 종류를 파악합니다. "변환" 버튼을 누르면 즉시 결과가 표시됩니다.

### 4단계: 결과 확인 및 저장
변환된 텍스트가 오른쪽 출력창에 깔끔하게 나타납니다. 한글이 제대로 보이는지 확인한 후, 복사하거나 파일로 다운로드하여 저장하세요. 이제 **한글 깨짐 복원**이 완료되었습니다.

## 주의사항: 인코딩 변환 시 꼭 체크할 점

WooaText는 강력하지만, 몇 가지 주의할 점이 있습니다.

- **원본 인코딩을 정확히 아는 경우 수동 선택** : 자동 감지가 항상 100% 정확하지는 않으므로, 원본 인코딩을 알고 있다면 직접 선택하는 것이 더 안전합니다.
- **특수 문자 포함 시 재확인** : 일부 특수 문자나 이모티콘은 변환 후 깨질 수 있으니 반드시 결과를 다시 확인하세요.
- **이미지나 바이너리 파일은 변환 불가** : 이 도구는 순수 텍스트만 처리합니다. 이미지 속 텍스트는 OCR 도구가 필요합니다.
- **파일 크기 제한** : 대용량 파일은 브라우저 성능에 따라 느려질 수 있으니 10MB 이하 파일 사용을 권장합니다.
