필터를 적용하려는 열의 헤더에 있는 화살표 버튼을 클릭합니다. 예를 들어 '주문일'이나 '등록일'과 같은 날짜 데이터가 들어간 열을 선택하세요. 그러면 필터 옵션 창이 열립니다.

### 2단계: 날짜 조건 입력

필터 옵션 창에서 '날짜 필터' 항목을 찾습니다. 여기서 시작 날짜와 종료 날짜를 직접 입력하거나 달력 아이콘을 클릭하여 원하는 날짜를 선택합니다. 예를 들어 2024년 1월 1일부터 2024년 1월 31일까지의 데이터만 보고 싶다면 해당 날짜를 각각 입력하면 됩니다.

날짜 조건을 설정한 후 '적용' 버튼을 누르면 즉시 해당 날짜 범위에 맞는 행만 화면에 표시됩니다. 필터가 적용된 상태에서 데이터를 확인하거나 수정할 수 있으며, 원본 파일은 그대로 유지됩니다.
