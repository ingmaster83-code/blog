### 1단계: 텍스트 입력하기
먼저 정렬하고 싶은 텍스트를 준비합니다. 각 항목은 줄바꿈(Enter)으로 구분되어 있어야 합니다. 예를 들어, 다음과 같은 과일 이름 목록이 있다고 가정해보겠습니다.
```

딸기
포도
```
이제 이 텍스트를 **WooaText** 웹사이트의 입력창에 붙여넣습니다.

### 2단계: 오름차순 정렬 버튼 클릭
텍스트를 입력한 후, 도구 상자에서 '정렬' 기능을 찾습니다. 여기서 '오름차순' 옵션을 선택하고 버튼을 클릭하면, 텍스트가 즉시 가나다순(또는 알파벳순)으로 재배열됩니다. 위의 예시를 실행하면 아래와 같이 변합니다.
```

포도
```
이렇게 간단합니다. **WooaText**는 전체 리스트를 깔끔하게 정리해주어 데이터 분석이나 문서 작업 효율을 높여줍니다.

### 3단계: 결과 복사 및 활용
정렬이 완료되면 결과 텍스트를 바로 복사하여 워드 문서, 이메일, 또는 블로그에 붙여넣으면 됩니다. **텍스트 정렬 오름차순 작업**이 이렇게 쉬울 수가 없습니다. 만약 순서를 반대로 하고 싶다면 '내림차순' 버튼만 클릭하면 됩니다.
