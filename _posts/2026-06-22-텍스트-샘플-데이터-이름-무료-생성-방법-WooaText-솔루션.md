## WooaText: 텍스트 샘플 데이터 이름 무료 생성 도구

[WooaText](https://textkit.wooahouse.com)는 설치나 회원가입 없이 브라우저에서 바로 사용할 수 있는 무료 온라인 텍스트 생성 도구입니다. 한국어 이름, 영문 이름, 주소, 전화번호, 이메일 등 **테스트 데이터 생성**에 필요한 모든 항목을 지원합니다. 특히 개인정보 보호에 민감한 작업에서 실제 데이터 대신 사용할 수 있는 **가짜 개인정보**를 즉시 만들어 줍니다.

### WooaText의 주요 장점

- **완전 무료**: 숨은 유료 결제나 크레딧 제한이 없습니다.
- **브라우저 기반**: 설치 불필요, 모든 운영체제에서 호환됩니다.
- **다양한 데이터 유형**: 이름뿐만 아니라 주소, 전화번호, 이메일, 회사명까지 생성 가능합니다.
- **개수 조절 가능**: 1개부터 수백 개까지 필요한 만큼 한 번에 생성합니다.

### WooaText의 단점

- **인터넷 필요**: 오프라인에서는 사용할 수 없습니다.
- **데이터 내보내기 제한**: 파일 다운로드 기능이 없어 복사 후 직접 저장해야 합니다.
- **고급 커스터마이징 부족**: 특정 성별이나 지역 기반 필터링은 지원되지 않습니다.

## WooaText 사용법 단계별 가이드

### 1. 도구 접속 및 기본 설정

[WooaText](https://textkit.wooahouse.com) 페이지에 접속하면 직관적인 인터페이스가 나타납니다. 상단 메뉴에서 '샘플 데이터 생성' 섹션을 선택하세요.

### 2. 데이터 유형 선택

**데이터 항목**에서 필요한 필드를 체크합니다. 기본적으로 이름, 성별, 나이, 주소, 전화번호, 이메일이 제공됩니다. **더미 데이터 만들기** 작업 시 주소나 이메일이 필요 없다면 체크를 해제하면 됩니다.

### 3. 생성 개수 설정

'생성 개수' 입력란에 원하는 숫자를 입력합니다. 10개, 50개, 100개 등 자유롭게 조정 가능합니다. 대량의 **랜덤 데이터 생성**이 필요할 때 유용합니다.

### 4. 결과 확인 및 복사

'생성하기' 버튼을 클릭하면 즉시 결과가 표시됩니다. 결과는 개별 행으로 정리되어 나오며, 전체를 선택하여 복사(Ctrl+C)한 후 원하는 문서에 붙여넣기(Ctrl+V) 하면 됩니다.
