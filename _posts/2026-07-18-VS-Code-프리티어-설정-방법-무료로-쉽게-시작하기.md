## 주요 기능 3가지, 실제로 써보니

### 1. 프리티어 설정 템플릿 제공
프리티어를 처음 쓸 때 `trailingComma`, `singleQuote` 같은 옵션 이름만 봐도 헷갈렸습니다.  
WooaVS는 **프리티어 설정 방법**을 단계별 템플릿으로 정리해줘서, 원하는 스타일을 클릭만으로 골라 쓸 수 있습니다.  
예를 들어 "세미콜론 필수" 옵션 하나만 켜도 코드가 일관성 있게 변합니다.

### 2. VS Code 확장팩 추천 및 다운로드
코드 포맷터만으로는 부족할 때가 많습니다. ESLint, Tailwind CSS IntelliSense 같은 확장팩이 필요하죠.  
이 도구는 **VS Code 설정**에 자주 쓰이는 확장팩을 추천하고, 각각의 공식 다운로드 링크를 제공합니다.  
한 페이지에서 모든 것을 해결할 수 있어서 검색 시간을 크게 줄여줍니다.

### 3. 프로젝트별 설정 파일 생성
리액트, Vue, Node.js 등 프레임워크마다 선호하는 코드 스타일이 다릅니다.  
WooaVS는 프로젝트 유형을 선택하면 자동으로 `.prettierrc` 파일을 생성해줍니다.  
저는 이 기능 덕분에 **코드 정리 자동화**를 30초 만에 완료했습니다.

## 활용 팁, 더 스마트하게 쓰는 법

프리티어를 잘 활용하려면 **자동 저장 시 포맷팅**을 꼭 켜두세요.  
VS Code 설정(`settings.json`)에 아래 한 줄만 추가하면 됩니다:

```json
"editor.formatOnSave": true
```

이렇게 하면 `Ctrl + S`를 누를 때마다 코드가 자동으로 정리됩니다.  
또한 팀 프로젝트에서는 `.editorconfig` 파일을 함께 사용하면 더 좋습니다.  
들여쓰기와 줄바꿈 스타일까지 통일할 수 있어서 코드 리뷰 시간이 반으로 줄었습니다.

WooaVS를 활용하면 이런 설정들을 **복잡한 검색 없이** 한눈에 볼 수 있어서 특히 초보자에게 추천합니다.
