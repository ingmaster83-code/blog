## 시트 수식 입력 방법 완전 정복

수식은 단순히 계산을 자동화하는 도구입니다. 하나씩 배우면 누구나 쉽게 익힐 수 있습니다.

### 2단계: 기본 수식 입력 순서

#### 등호(=)부터 입력하세요

모든 수식은 반드시 등호로 시작합니다. 셀을 클릭한 후 키보드로 `=`를 누르면 수식 입력 모드가 활성화됩니다.

#### 함수 이름을 입력하거나 선택하세요

`=SU`까지 입력하면 자동 완성 목록이 나타납니다. `SUM`, `SUBTOTAL` 등 추천 함수가 표시되며, 마우스나 키보드 방향키로 선택할 수 있습니다. 이 자동 완성 기능은 초보자에게 특히 유용하며, 함수 철자를 외우지 않아도 됩니다.

#### 괄호 안에 범위를 지정하세요

예를 들어 `=SUM(A1:A10)`은 A1부터 A10까지의 값을 모두 더하라는 뜻입니다. 셀 범위는 마우스로 드래그하여 선택할 수도 있고, 직접 입력할 수도 있습니다. 구글 스프레드시트 함수와 동일한 방식으로 작동하므로 기존 사용자도 부담 없이 사용 가능합니다.

#### Enter 키로 완성하세요

수식을 다 입력한 후 Enter를 누르면 결과가 즉시 표시됩니다. 아래 셀에도 같은 수식을 적용하려면 셀 우측 하단 모서리를 잡고 드래그하면 자동으로 채워집니다.

## 주요 기능으로 업무 효율 높이기

### SUM 함수로 합계 구하기

가장 많이 사용되는 함수입니다. 매출 합계, 출석 인원 합계 등 숫자 데이터를 빠르게 더할 수 있습니다.
```
=SUM(B2:B20)
```

### AVERAGE 함수로 평균 계산

학생 성적 평균, 일일 평균 방문자 수 등 평균값이 필요할 때 사용합니다. 데이터 분석의 기초가 되는 함수입니다.
```
=AVERAGE(C2:C30)
```

### IF 함수로 조건 판단

조건에 따라 다른 값을 표시합니다. 예를 들어 점수가 80점 이상이면 "합격", 아니면 "불합격"으로 표시할 수 있습니다.
```
=IF(D2>=80, "합격", "불합격")
```

### COUNT 함수로 개수 세기

숫자가 입력된 셀의 개수를 셉니다. 빈 셀은 제외되므로 데이터 입력 현황을 파악할 때 유용합니다.
```
=COUNT(A1:A100)
```

## 실무에서 바로 써먹는 활용 팁

### 데이터 정리와 수식 결합

원시 데이터를 정리한 후 수식을 적용하면 업무 속도가 2배 이상 빨라집니다. 예를 들어 판매 데이터에서 특정 조건을 만족하는 행만 골라 합계를 구할 때는 SUMIF 함수를 활용하세요.
```
=SUMIF(A:A, "서울", B:B)
```

### 셀 참조 방식 이해하기

상대 참조($ 없음)는 수식을 복사할 때 자동으로 위치가 조정되고, 절대 참조($A$1)는 항상 같은 셀을 가리킵니다. 혼합 참조($A1 또는 A$1)도 상황에 따라 유용하게 사용됩니다. 이 개념만 알아도 수식 오류의 90%를 예방할 수 있습니다.

### 오류 처리와 디버깅

수식 결과에 `#DIV/0!`나 `#N/A` 같은 오류가 뜨면 당황하지 마세요. IFERROR 함수로 오류를 깔끔하게 처리할 수 있습니다.
```
=IFERROR(A1/B1, 0)
```
