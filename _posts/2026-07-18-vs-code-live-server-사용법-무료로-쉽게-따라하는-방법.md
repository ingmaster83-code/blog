## 주요 기능: 실시간 미리보기와 코드 자동 반영

### 1. 실시간 HTML 미리보기
WooaVS의 핵심 기능은 코드를 입력하면 즉시 결과물이 화면에 반영된다는 점입니다. 예를 들어, HTML 태그를 수정하면 0.1초 안에 우측 화면이 바뀌어서, 디자인 오류를 바로잡거나 레이아웃을 테스트하기에 최적화되어 있습니다.

### 2. CSS와 JavaScript 즉시 반영

### 3. 파일 저장 기능
