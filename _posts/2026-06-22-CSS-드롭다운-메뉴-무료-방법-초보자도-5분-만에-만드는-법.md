## 드롭다운 메뉴 만드는 단계별 사용법

### 1단계: 메뉴 항목 입력하기

사이트에 접속하면 상단 메뉴바, 서브 메뉴 구조를 입력할 수 있는 입력창이 나타납니다. 여기에 원하는 메뉴 이름을 텍스트로 적어 넣으면 됩니다. 예를 들어 "회사소개, 제품, 서비스, 문의하기" 같은 대메뉴와 그 아래에 들어갈 하위 메뉴를 계층 구조로 입력합니다.

### 2단계: 디자인과 스타일 선택

메뉴 구조를 입력한 후에는 스타일을 선택합니다. 기본적인 세로형 또는 가로형 드롭다운 레이아웃을 고를 수 있고, 배경색, 글자색, 호버 효과 색상까지 간단히 설정할 수 있습니다. 이 단계에서 미리보기가 실시간으로 업데이트되므로 원하는 디자인이 나올 때까지 조정하면 됩니다.

### 3단계: 코드 생성 및 복사

스타일 설정이 끝났다면 "생성" 또는 "코드 복사" 버튼을 클릭합니다. 그러면 즉시 완성된 HTML 코드와 CSS 코드가 텍스트 영역에 나타납니다. 이 코드를 전체 선택하여 복사한 후, 자신의 웹사이트 HTML 파일의 `<head>` 부분에 CSS를, `<body>` 부분에 HTML 구조를 붙여넣으면 끝입니다.

### 4단계: 실제 웹페이지에 적용하기

복사한 코드를 웹페이지 소스에 붙여넣고 저장한 후 브라우저에서 확인해보면 깔끔하게 작동하는 CSS 드롭다운 메뉴를 볼 수 있습니다. 호버 시 하위 메뉴가 부드럽게 펼쳐지는 애니메이션까지 기본 포함되어 있어서 따로 추가 작업이 필요 없습니다.

## CSS 메뉴 생성 시 주의사항

이 도구가 매우 편리하긴 하지만, 몇 가지 점을 미리 알아두면 더 좋습니다.

**1. 메뉴 깊이 제한**  
WooaDev는 일반적인 2단계 드롭다운(대메뉴-서브메뉴)을 기본 지원합니다. 3단계 이상의 깊은 계층이 필요하면 생성된 코드를 약간 수정해야 할 수 있습니다.

**2. CSS 클래스 충돌 확인**  
생성된 코드에는 고유한 CSS 클래스 이름이 붙어 있지만, 기존 사이트에 이미 같은 이름의 클래스가 있다면 스타일이 겹칠 수 있습니다. 붙여넣기 전에 클래스 이름을 한번 확인하는 것이 좋습니다.

**3. 반응형 추가 작업**  
기본 코드는 데스크탑에 최적화되어 있습니다. 모바일 화면에서 햄버거 메뉴로 바꾸거나 터치 이벤트를 추가하려면 약간의 미디어 쿼리 작업이 필요할 수 있습니다. 하지만 기본적인 드롭다운 동작 자체는 모바일에서도 문제없이 작동합니다.

**4. 자바스크립트 불필요**  
이 도구로 생성된 메뉴는 순수 CSS만으로 드롭다운이 구현되기 때문에 자바스크립트를 따로 추가할 필요가 없습니다. 로딩 속도가 빠르고 유지보수가 쉽다는 장점이 있습니다.
