## VS 코드 Git 연동의 주요 기능

### Git 저장소 자동 연결 및 경로 설정

WooaVS는 Git이 설치된 위치를 자동 탐색하고, VS Code의 설정 파일에 경로를 추가합니다. 기존에는 `git.path`를 수동으로 입력하거나 환경변수를 편집해야 했지만, 이 도구는 버튼 한 번으로 모든 설정을 완료합니다. 또한 VS Code 터미널이 Git Bash를 기본 셸로 인식하도록 도와줘, 명령어 입력 오류를 줄여줍니다.

### 충돌 해결 시각화와 커밋 관리

Git 연동에서 가장 까다로운 부분은 병합 충돌(merge conflict)입니다. WooaVS는 충돌이 발생한 파일을 별도 창에 표시하고, 각 변경 사항을 색상으로 구분해줍니다. '내 변경 유지', '원격 변경 유지', '둘 다 유지' 같은 옵션을 클릭만으로 선택할 수 있어, 코드를 직접 수정하지 않아도 됩니다. 또한 커밋 메시지 템플릿을 제공해 일관된 기록을 남기도록 도와줍니다.

### 원격 저장소 연동 오류 진단

깃허브나 깃랩과 연결할 때 자주 발생하는 'Permission denied (publickey)' 오류를 자동으로 진단합니다. SSH 키 생성부터 등록까지 단계별 가이드를 제공하며, 잘못된 리모트 URL을 감지해 수정 제안을 합니다. 이를 통해 VS 코드 Git 연동 시 인증 문제로 멈추는 일이 없습니다.
