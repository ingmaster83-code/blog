## 핵심 기능 상세: 페이지 바로 찾기

### 1. 페이지 번호 직접 입력하기

가장 강력한 기능은 **페이지 번호 입력을 통한 즉시 이동**입니다. 상단 툴바에 "45"를 입력하고 Enter를 누르면, 로딩 없이 바로 45페이지로 점프합니다.

- **적용 가능한 파일 형식**: PDF, EPUB, 이미지 시퀀스 등
- **입력 방식**: 숫자만 입력하면 되는 직관적인 UI
- **반응 속도**: 0.5초 이내 페이지 전환

### 2. 키보드 단축키 활용하기

생산성을 더 높이고 싶다면 키보드 단축키를 익혀보세요. 마우스를 건드릴 필요 없이 페이지 이동을 실행할 수 있습니다.

- **좌우 방향키**: 이전/다음 페이지
- **Ctrl + G**: 페이지 이동 입력창 바로 열기
- **Home / End**: 문서 처음과 끝으로 이동

이 단축키들은 **파일 특정 페이지 이동 방법**을 찾는 사람이라면 반드시 알아야 할 핵심 테크닉입니다.

### 3. 검색 기능으로 텍스트 기반 이동

특정 내용이 포함된 페이지로 이동하고 싶다면 내부 검색 기능(Ctrl+F)을 활용하세요. '계약서'라는 단어를 검색하면 해당 단어가 포함된 모든 페이지를 목록으로 보여주고, 클릭 한 번으로 이동할 수 있습니다.
