## WooaDev의 주요 기능

### CSS 변수 자동 생성
WooaDev는 입력한 색상 값을 기반으로 다크모드 CSS 변수를 자동 생성합니다. 배경색, 글자색, 강조색 등 주요 색상만 입력하면 나머지는 도구가 알아서 계산합니다. 이 과정에서 대비 비율을 고려해 접근성까지 자동으로 보정해줍니다.

### 실시간 미리보기
코드를 생성하기 전에 실시간 미리보기 기능을 통해 라이트 모드와 다크 모드가 어떻게 보일지 즉시 확인할 수 있습니다. 색상을 조정할 때마다 미리보기가 업데이트되므로 원하는 디자인을 빠르게 찾을 수 있습니다.

### 코드 복사 및 내보내기
생성된 CSS 코드는 한 번의 클릭으로 클립보드에 복사할 수 있습니다. HTML 파일에 직접 붙여넣거나, 별도의 CSS 파일로 저장할 수 있어 다양한 웹 프로젝트에 바로 적용 가능합니다.
