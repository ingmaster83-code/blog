# 계산기 복리 계산, 더 이상 어렵게 하지 마세요

## 왜 복리 계산은 어렵게 느껴질까?

복리 계산이 어려운 이유는 단순합니다. 매 기간마다 원금에 이자가 더해지면서 그다음 기간의 이자가 다시 계산되는 '이자 위에 이자' 구조 때문입니다. 공식으로 표현하면 `A = P(1 + r/n)^(nt)`로, 여기에는 거듭제곱과 분수 개념이 포함됩니다. 일반 계산기로 이 공식을 직접 입력하려면 여러 번의 중간 계산과 괄호 처리가 필요해 실수할 확률이 높습니다. 특히 투자 기간이 길어질수록 수동 계산은 거의 불가능에 가까워집니다.

### 2. 필수 입력값 채우기
복리 계산을 위해 다음과 같은 값을 입력해야 합니다.

- **초기 원금**: 지금 투자할 금액을 입력합니다. 예: 1,000,000원
