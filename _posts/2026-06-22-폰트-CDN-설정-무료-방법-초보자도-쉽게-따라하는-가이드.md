웹사이트를 운영하다 보면 예쁜 폰트를 적용하고 싶지만, 용량이 커서 속도가 느려질까 봐 고민한 적이 있으신가요? 또는 `@font-face`를 사용해 폰트 파일을 직접 서버에 올리는 과정이 복잡해서 포기한 경험이 있으실 겁니다.

이런 고민을 한방에 해결해 줄 **폰트 CDN 설정 무료 방법**을 소개합니다. 서버에 파일을 업로드할 필요 없이, 단 몇 줄의 코드만 추가하면 원하는 웹폰트를 즉시 사용할 수 있습니다.

## 무료 폰트 CDN 설정 도구 FontKit 소개

직접 폰트를 서버에 올리면 트래픽 부담이 생기고, 라이선스 문제도 신경 써야 합니다. 하지만 CDN(Content Delivery Network)을 활용하면 이런 문제가 모두 해결됩니다.

**FontKit**([WooaHouse FontKit](https://fontkit.wooahouse.com))은 누구나 쉽게 웹폰트를 적용할 수 있도록 도와주는 무료 도구입니다. 구글 폰트, 눈누 폰트 등 다양한 무료 폰트를 지원하며, 원하는 폰트를 선택하면 자동으로 CDN 링크와 CSS 코드를 생성해 줍니다.

### FontKit의 주요 장점
- **설치 불필요** : 회원가입이나 프로그램 설치 없이 웹사이트에서 바로 사용 가능
- **속도 최적화** : 글로벌 CDN을 통해 빠른 로딩 속도 보장
- **웹사이트 최적화** : 웹폰트 적용으로 사이트 디자인 완성도 향상

## 폰트 CDN 설정 단계별 따라하기

이제 본격적으로 FontKit을 사용하여 웹폰트를 적용해 보겠습니다. 과정이 매우 간단하니 차근차근 따라오세요.

### 1단계: FontKit 접속 및 폰트 검색
먼저 [FontKit](https://fontkit.wooahouse.com)에 접속합니다. 사이트에 들어가면 검색창이 보입니다. 원하는 폰트 이름(예: "Noto Sans KR", "GmarketSans")을 입력하거나, 인기 폰트 목록에서 마음에 드는 것을 선택하세요.

### 2단계: 폰트 스타일과 굵기 선택
폰트를 클릭하면 상세 페이지로 이동합니다. 여기서 **웹폰트 적용**에 필요한 옵션을 선택합니다.
- **Regular(400), Bold(700)** 등 원하는 폰트 굵기를 체크하세요.
- 필요한 굵기만 선택할수록 CSS 파일 용량이 줄어들어 사이트 속도에 유리합니다.

### 3단계: CDN 코드 생성 및 복사
스타일을 선택하면 오른쪽에 코드 생성 영역이 활성화됩니다. 여기서 두 가지 코드가 제공됩니다.

**1) CSS 코드(HTML head에 추가)**
```html
<link href="[https://cdn.wooahouse.com/css2?family=Noto+Sans+KR:wght@400;700&display=swap"](https://cdn.wooahouse.com/css2?family=Noto+Sans+KR:wght@400;700&display=swap") rel="stylesheet">
```

**2) CSS 속성(style.css에 추가)**
```css
font-family: 'Noto Sans KR', sans-serif;
```

### 4단계: 웹사이트에 코드 적용
이제 생성된 코드를 웹사이트에 붙여넣기만 하면 됩니다.

- **워드프레스 사용자** : `테마 디자인` > `테마 파일 편집기`에서 `header.php` 파일의 `<head>` 태그 안에 CSS 링크를 추가하세요.
- **HTML 직접 제작** : 모든 페이지의 `<head>` 태그 안에 링크를 추가한 후, CSS 파일에서 `font-family` 속성을 적용합니다.
- **티스토리, 블로그** : 스킨 편집의 HTML 탭에서 `<head>` 부분을 찾아 코드를 삽입하면 됩니다.

### 5단계: 적용 확인
웹사이트를 새로고침하여 폰트가 정상적으로 적용되었는지 확인합니다. 만약 폰트가 깨지거나 적용되지 않는다면, **CDN 링크**가 정확히 복사되었는지, `<link>` 태그가 깨지지 않았는지 다시 한번 점검해 보세요.

## 웹폰트 적용 시 주의사항

폰트 CDN 설정은 쉽지만, 몇 가지 주의할 점이 있습니다.

- **라이선스 확인** : 무료 폰트라도 상업적 사용이 제한되는 경우가 있습니다. 반드시 해당 폰트의 라이선스를 확인하세요.
- **폰트 굵기 최소화** : 400(일반)과 700(굵게) 정도만 선택하는 것이 로딩 속도에 좋습니다. 불필요하게 많은 굵기를 선택하면 페이지 로딩이 느려질 수 있습니다.
- **캐시 활용** : CDN을 사용하면 방문자가 한 번 다운로드한 폰트는 브라우저에 캐시되어, 이후 방문 시 더 빠르게 로딩됩니다.
- **대체 폰트 설정** : `font-family` 속성 마지막에 `sans-serif`나 `serif` 같은 기본 폰트를 꼭 넣어주세요. CDN이 일시적으로 다운되더라도 기본 폰트가 적용되어 사이트 가독성이 유지됩니다.
