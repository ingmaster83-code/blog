## WooaText 주요 무료 기능 3가지

WooaText는 텍스트 작업에서 가장 자주 사용하는 기능만 모아놓은 실용적인 온라인 텍스트 변환 도구입니다. 핵심 기능은 크게 세 가지로 나뉩니다.

### 1. 중복 줄 제거 기능

데이터 수집이나 로그 분석 시 중복된 텍스트 줄을 빠르게 걸러냅니다.

- **적용 예시**: 이메일 주소 목록, 제품 코드 리스트, 설문조사 응답 데이터
- **사용 효과**: 수백 줄의 데이터도 1초 만에 유니크한 값만 추출

### 2. 공백 및 줄바꿈 정리 기능

문서 간 복사/붙여넣기 시 발생하는 여분의 공백과 줄바꿈을 일괄 정리합니다.

- **적용 예시**: PDF에서 복사한 계약서 본문, 웹페이지에서 긁어온 기사 텍스트
- **사용 효과**: 텍스트 포맷을 깔끔하게 통일하여 가독성 향상

### 3. 대소문자 일괄 변환 기능

영문 텍스트의 대소문자를 원하는 형식으로 한 번에 변경합니다.

- **지원 형식**: 모두 대문자, 모두 소문자, 첫 글자만 대문자, 각 단어 첫 글자 대문자
- **적용 예시**: 제목 포맷 통일, 코드 주석 정리, 영문 보고서 일관성 유지

## WooaText 사용법: 단계별 실전 가이드

실무에서 바로 따라 할 수 있도록 상세 단계로 설명합니다.

### 단계 1: 웹사이트 접속하기

브라우저를 열고 [WooaText 공식 페이지](https://textkit.wooahouse.com)에 접속합니다. 별도의 설치나 회원가입 없이 메인 화면에 텍스트 입력 영역이 바로 나타납니다.

### 단계 2: 원본 텍스트 붙여넣기

정리할 텍스트를 상단 입력창에 붙여넣습니다. 엑셀, 워드, PDF, 웹페이지 등 어떤 소스에서 복사해도 무방합니다. 페이지 좌측에는 미리보기 영역이 있어 작업 전후를 실시간으로 비교할 수 있습니다.

### 단계 3: 원하는 기능 선택 후 실행

입력창 아래에 있는 기능 버튼 중 필요한 것을 클릭합니다.

- **중복 줄 제거**: 중복되는 라인이 있다면 모두 제거
- **공백 정리**: 앞뒤 공백, 연속 공백, 과도한 줄바꿈을 정리
- **대소문자 변환**: 원하는 대소문자 형식 선택
- **특정 문자 치환**: 찾아 바꾸기 기능으로 특정 단어나 기호 변경

각 버튼을 누르면 즉시 결과가 우측 출력창에 표시됩니다. 여러 기능을 순차적으로 적용할 수도 있습니다.

### 단계 4: 결과 복사 및 저장

정리된 텍스트를 확인한 후 출력창 하단의 복사 버튼을 클릭하거나 직접 선택하여 복사합니다. 그대로 문서, 이메일, 메모장 등에 붙여넣어 사용하면 됩니다.
