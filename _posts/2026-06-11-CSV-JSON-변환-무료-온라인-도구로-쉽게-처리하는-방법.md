### WooaSheet 주요 기능 상세

#### CSV를 JSON으로 변환하는 핵심 기능

이 도구의 가장 큰 장점은 CSV 파일을 JSON 배열 형태로 자동 변환해준다는 점입니다. 각 행이 하나의 객체가 되고, 헤더 행이 키(Key)로 지정되기 때문에 개발자에게 바로 전달할 수 있는 깔끔한 구조가 만들어집니다. 예를 들어 고객명, 이메일, 전화번호가 적힌 CSV라면 다음과 같은 JSON이 출력됩니다.

```json
[
  {"고객명": "홍길동", "이메일": "test@example.com", "전화번호": "010-1234-5678"},
  {"고객명": "김철수", "이메일": "kim@example.com", "전화번호": "010-9876-5432"}
]
```

이런 식으로 '데이터 구조 변환'이 자동으로 이루어지기 때문에 일일이 손으로 수정할 필요가 전혀 없습니다.

#### JSON to CSV 변환도 가능한 양방향 지원

반대로 개발자로부터 JSON 데이터를 받아서 엑셀이나 구글 시트에서 분석해야 할 때도 이 도구가 유용합니다. JSON 파일을 업로드하면 CSV 형식으로 다운로드할 수 있어 바로 표 형태로 확인할 수 있습니다. '온라인 포맷 변환' 툴이지만 속도가 매우 빠르고, 파일 업로드 용량 제한도 없는 편이라 실무에서 부담 없이 사용할 수 있었습니다.
