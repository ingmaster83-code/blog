### 주요 기능

#### 브랜치 관리 명령어 모음
브랜치 생성, 전환, 삭제, 병합까지 필수 명령어를 그룹화했습니다. `git merge`와 `git rebase`의 차이도 바로 확인할 수 있습니다.

#### 커밋 및 되돌리기 명령어
`git commit --amend`나 `git reset`처럼 실수했을 때 빠르게 대처하는 명령어를 별도 섹션으로 제공합니다. 초보자도 실무에서 바로 응용할 수 있습니다.

#### 원격 저장소 명령어
팀 협업 시 필요한 `git push`, `git pull`, `git fetch`의 정확한 사용법과 옵션을 정리했습니다. Git 협업 워크플로를 더 효율적으로 만들어 줍니다.

### 활용 팁

**치트시트를 실무에 바로 적용하는 3단계**
1. **자주 실수하는 명령어 위주로 학습**하세요. 예를 들어 `git stash`와 `git restore`를 구분하지 못했다면 해당 부분을 먼저 익힙니다.
2. **브라우저를 항상 열어두고 참고**하세요. WooaDev는 반응형 웹이므로 모바일에서도 확인 가능합니다.
3. **출력해서 책상에 붙여두세요.** 브라우저 출력 기능으로 A4 한 장에 핵심만 정리된 Git 명령어 치트시트를 만들 수 있습니다.
