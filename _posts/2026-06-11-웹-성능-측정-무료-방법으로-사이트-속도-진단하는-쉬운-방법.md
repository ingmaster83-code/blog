## 사용법 단계별

### 1. 사이트 URL 입력하기

먼저 [WooaDev](https://wooadev.wooahouse.com)에 접속합니다. 메인 화면 중앙에 보이는 입력창에 자신이 운영하는 웹사이트 주소를 붙여넣기 하세요. 예를 들어 블로그 메인 페이지나 랜딩 페이지의 URL을 입력하면 됩니다.

### 2. 분석 시작 버튼 클릭

입력 후 옆에 있는 시작 버튼을 누르면 자동으로 웹 성능 측정이 시작됩니다. 이 과정에서 도구는 서버 응답 시간, CSS/JS 파일 로딩 속도, 이미지 최적화 상태 등을 한 번에 체크합니다. 결과는 20~40초 내에 완료됩니다.

### 3. 결과 해석하기

분석이 끝나면 점수와 함께 상세 보고서가 나타납니다. 여기서 주목할 항목은 세 가지입니다:
- **총 로딩 시간**: 사용자가 체감하는 속도
- **리소스 크기**: 너무 큰 이미지나 스크립트가 없는지 확인
- **요청 수**: 불필요한 HTTP 요청이 많은지 파악

개선이 필요한 부분은 빨간색이나 주황색으로 표시되므로 바로 조치할 수 있습니다.
