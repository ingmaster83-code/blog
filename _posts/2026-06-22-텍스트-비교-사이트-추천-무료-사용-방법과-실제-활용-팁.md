## 텍스트 비교 방법 단계별 가이드

### 1단계: WooaText 접속하기

웹 브라우저를 열고 [WooaText 페이지](https://textkit.wooahouse.com)로 이동합니다. 별도 로그인이나 다운로드 절차가 없어 즉시 메인 화면이 나타납니다. 좌측과 우측에 각각 텍스트를 입력할 수 있는 빈 공간이 준비되어 있습니다.

### 2단계: 비교할 텍스트 입력하기

**파일 업로드 방식**:
- 좌측 상단의 파일 선택 버튼을 클릭해 원본 문서를 업로드합니다.
- 우측 상단의 파일 선택 버튼으로 수정된 문서를 업로드합니다.
- 지원 형식은 TXT, CSV, HTML이며 최대 1MB까지 가능합니다.

**직접 붙여넣기 방식**:
- 좌측 입력창에 원본 텍스트를 붙여넣습니다.
- 우측 입력창에 비교 대상 텍스트를 붙여넣습니다.
- 코드 블록이나 줄 바꿈이 포함된 문서도 그대로 유지됩니다.

### 3단계: 비교 실행 및 결과 확인

입력이 완료되면 화면 하단의 비교하기 버튼을 클릭합니다. 결과 화면에서는 다음과 같이 시각적으로 차이가 표시됩니다.
- **초록색 배경**: 새롭게 추가된 줄이나 단어
- **빨간색 배경**: 삭제된 줄이나 단어
- **회색 배경**: 변경 없이 동일한 부분

변경된 부분만 모아보는 필터 기능도 지원해 전체 문서를 다시 훑지 않고 핵심 변경점만 집중해서 검토할 수 있습니다.

## 텍스트 비교 시 주의사항

텍스트 비교 도구를 사용할 때 몇 가지 점을 유의하면 더 정확한 결과를 얻을 수 있습니다.

1. **일관된 줄바꿈 유지**: 한쪽 문서는 윈도우 스타일(CRLF), 다른 쪽은 맥 스타일(LF)이면 차이로 인식될 수 있습니다. 가능하면 동일한 환경에서 저장한 문서를 비교하세요.
2. **공백 문자 확인**: 문장 끝의 여분 공백이나 탭 문자도 차이로 표시되므로, 의미 없는 공백은 사전에 제거하는 것이 좋습니다.
3. **대소문자 구분**: 코드나 데이터 비교 시 대소문자가 중요한 경우가 많지만, 일반 문서 비교에서는 대소문자 무시 옵션을 활용하면 더 효과적입니다.
4. **인코딩 일치**: UTF-8과 EUC-KR 등 서로 다른 인코딩으로 저장된 파일은 깨짐 현상이 발생할 수 있으므로, 비교 전에 동일한 인코딩으로 통일하세요.
