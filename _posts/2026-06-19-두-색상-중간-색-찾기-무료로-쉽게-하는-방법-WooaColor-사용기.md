## 주요 기능: 두 색상의 정확한 중간값을 실시간으로 확인

### 1. 두 가지 색상 선택하기
화면에 보이는 두 개의 색상 선택 패널에서 각각 원하는 색상을 고르면 됩니다. 저는 메인 브랜드 컬러인 네이비 블루(#1A237E)와 서브 컬러인 코랄 핑크(#FF6F61)를 선택해봤습니다. 색상을 고르는 순간 실시간으로 중간 색상이 생성되기 때문에 굉장히 직관적입니다.

### 2. HEX 코드와 RGB 값 자동 표시
생성된 중간 색상은 화면 중앙에 큰 원형으로 표시되며, 동시에 해당 색상의 HEX 코드와 RGB 값이 함께 나타납니다. 이 값들을 복사해서 피그마나 포토샵, CSS 파일 등에 바로 붙여넣기 할 수 있어 매우 편리했습니다. **색상 혼합 도구**로서의 기본 기능에 충실하면서도 실용성을 놓치지 않은 점이 마음에 듭니다.

### 3. 다양한 컬러 팔레트 실험
한 번의 설정으로 끝나는 것이 아니라, 슬라이더를 조정하거나 다른 두 색상을 다시 선택하면 즉시 새로운 중간색이 계산됩니다. 덕분에 여러 **색 조합 사이트**에서 제공하는 추천 팔레트보다도 더 세밀하게 내 의도에 맞는 중간색을 찾을 수 있었습니다. 여러 색상을 연속해서 비교해보고 싶은 분들에게 특히 유용합니다.
