### 주요 기능: 키 알파벳순 정렬과 가독성 향상

WooaText의 JSON 키 정렬 기능은 한 가지 목표에 집중합니다. 입력된 JSON 객체의 모든 키를 알파벳순(A-Z)으로 재배치하는 것입니다. 중첩된 객체(Nested Object) 내부의 키까지 재귀적으로 정렬해 주므로 복잡한 구조도 일관된 순서로 확인할 수 있습니다. 예를 들어 `{ "z": 1, "a": 2, "c": { "b": 3, "d": 4 } }`를 입력하면 `{ "a": 2, "c": { "b": 3, "d": 4 }, "z": 1 }`로 변환됩니다. JSON 포맷터 기능이 함께 적용되어 들여쓰기도 자동으로 정리되므로 가독성이 한층 올라갑니다.

### 사용 방법: 3단계로 끝내기

1. [WooaText](https://textkit.wooahouse.com)에 접속합니다.
2. 좌측 입력창에 정렬할 JSON 데이터를 붙여넣습니다.
3. 우측 출력창에서 정렬된 결과를 확인하고 복사합니다.

### 실제 사용 후기: 생산성을 높여준 경험

WooaText를 발견한 후에는 JSON을 붙여넣고 정렬 버튼을 누르는 데 10초도 걸리지 않습니다. 특히 코드 리뷰 시 diff 툴과 함께 사용하면 변경된 키만 빠르게 캐치할 수 있어서 효율이 배가되었습니다.
