## 주요 기능과 사용법

### 기본 팔레트 생성 방식

사이트에 접속하면 화면 중앙에 5개의 색상 원이 표시됩니다. 각 원은 클릭 한 번으로 색상 선택기가 열리며, 원하는 RGB, HEX, HSL 값을 직접 입력하거나 색상 맵에서 선택할 수 있습니다. 랜덤 생성 버튼을 누르면 알고리즘이 자동으로 조화로운 색상 조합을 추천해주므로, 아이디어가 떠오르지 않을 때 특히 유용합니다.

### 색상 고정과 비교 기능

마음에 드는 색상을 찾았다면, 해당 원 아래 있는 자물쇠 아이콘을 클릭하세요. 잠긴 색상은 새 팔레트를 생성해도 변하지 않고 유지됩니다. 예를 들어 브랜드 메인 컬러를 고정해두고, 서브 컬러만 계속 바꿔가며 조합을 테스트할 수 있습니다. 이 기능은 색상 대비와 보색 관계를 실험할 때 큰 도움이 됩니다.

### 팔레트 공유와 내보내기

생성한 팔레트는 하단의 공유 버튼으로 고유 링크를 만들 수 있습니다. 이 링크를 팀원이나 클라이언트에게 보내면, 상대방도 동일한 팔레트를 즉시 확인할 수 있습니다. 또한 색상 코드는 HEX 값으로 표시되며, 각 색상 위에 마우스를 올리면 자동 복사 버튼이 나타나 편리하게 추출할 수 있습니다. 이처럼 **무료 색상 추출** 기능은 웹 디자인과 UI/UX 작업에 필수적입니다.

## 실제 사용 후기와 장단점

### 장점

가장 큰 장점은 **심플함**입니다. 기능이 너무 많아 오히려 헷갈리는 다른 도구들과 달리, WooaColor는 필요한 것만 깔끔하게 제공합니다. 팔레트 생성, 색상 고정, 공유라는 세 가지 핵심 기능에 집중되어 있어 초보자도 부담 없이 사용할 수 있습니다. 또한 모든 작업이 브라우저에서 실시간으로 이루어지므로, 속도도 빠르고 데이터 사용량도 적습니다.

### 단점

다만, 고급 사용자에게는 기능이 다소 제한적일 수 있습니다. 예를 들어 팔레트를 이미지 파일로 내보내는 기능이나, 이미지에서 색상을 추출하는 **색상 추출기** 기능은 없습니다. 또한 색상 수를 5개로 고정할 수 없고, 기본이 5색이라는 점이 아쉽습니다. 3색이나 4색 팔레트가 필요하면 직접 색상을 삭제해야 합니다.

### 총평

가벼운 색상 조합 실험이나 빠른 프로토타이핑에 최적화된 도구입니다. 특히 SNS 콘텐츠 제작, 간단한 로고 디자인, 또는 인테리어 색상 기획처럼 복잡하지 않은 작업에 안성맞춤입니다. 전문 디자이너보다는 초보자나 비전공자에게 더 추천하고 싶은 **온라인 컬러 도구**입니다.
