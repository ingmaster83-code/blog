## SUMIF 함수 주요 기능과 사용법

SUMIF는 **조건부 합계 함수**로, 지정한 범위에서 특정 조건을 만족하는 셀만 골라서 더해줍니다. 기본 구조는 다음과 같습니다.

> `=SUMIF(조건 범위, 조건, 합계 범위)`

- **조건 범위**: 조건을 검사할 셀의 영역
- **조건**: 더하고 싶은 값을 걸러낼 기준 (텍스트, 숫자, 셀 참조 등)
- **합계 범위**: 실제로 더할 숫자가 있는 영역

### 예제: “서울” 지역 매출만 합산하기

1. A열에 지역(서울, 부산, 대구), B열에 매출 금액이 입력된 표가 있다고 가정합니다.
2. 결과를 표시할 셀에 아래 수식을 입력합니다.

```
=SUMIF(A2:A10, "서울", B2:B10)
```

3. Enter를 누르면 A2~A10 중 “서울”인 행의 B열 값만 더해집니다.

### 숫자 조건으로 합계 구하기

“5000 이상인 값만 더하기” 같은 작업도 쉽습니다. 조건에 `">=5000"`처럼 비교 연산자를 따옴표 안에 넣으면 됩니다.

```
=SUMIF(B2:B10, ">=5000", B2:B10)
```

이때 조건 범위와 합계 범위가 같으면 합계 범위를 생략할 수도 있습니다.

## 시트 함수 sumif 사용법 무료 연습을 위한 활용 팁

### 1. 와일드카드로 부분 일치 조건 만들기

조건에 `*`(별표)나 `?`(물음표)를 사용하면 특정 글자가 포함된 데이터를 찾을 수 있습니다. 예를 들어 “가”로 시작하는 모든 항목을 더하려면 조건을 `"가*"`로 입력하세요.

```
=SUMIF(A2:A10, "가*", B2:B10)
```

### 2. 셀 참조로 조건을 동적으로 변경하기

조건을 직접 입력하는 대신 셀을 참조하면 수식을 바꾸지 않고 조건만 변경할 수 있어 편리합니다. 예를 들어 C1 셀에 “서울”을 입력해두고 수식을 `=SUMIF(A2:A10, C1, B2:B10)`처럼 작성하면 C1 값만 바꿔서 다양한 조건의 합계를 빠르게 구할 수 있습니다.

### 3. SUMIFS와의 차이점 이해하기

SUMIF는 **단일 조건**만 처리합니다. 여러 조건(예: “서울이면서 3월 매출”)이 필요하면 SUMIFS 함수를 사용해야 합니다. 기본 구조는 비슷하지만 조건 범위와 조건 쌍이 여러 개 들어갑니다. WooaSheet에서 두 함수를 모두 무료로 테스트해보며 차이를 익혀보세요.
