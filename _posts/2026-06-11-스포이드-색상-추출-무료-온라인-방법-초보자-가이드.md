## WooaColor 소개: 무료 온라인 스포이드 도구

WooaColor([WooaColor 무료로 사용하기](https://colorkit.wooahouse.com))는 웹 브라우저에서 곧바로 사용할 수 있는 강력한 **온라인 컬러 피커**입니다. 별도의 설치나 로그인 없이, 이미지를 업로드하거나 웹페이지 화면에서 직접 원하는 지점의 색상을 클릭 한 번으로 캡처할 수 있습니다. 추출된 색상은 HEX, RGB, HSL 등 다양한 형식으로 즉시 확인할 수 있어, 개발자와 디자이너 모두에게 유용합니다.

## 주요 기능: 3단계로 끝내는 색상 추출

### 1단계: 이미지 업로드 또는 화면 캡처
[WooaColor](https://colorkit.wooahouse.com)에 접속하면 가장 먼저 두 가지 옵션이 보입니다. 하나는 '이미지 업로드'로 컴퓨터나 모바일에서 사진을 직접 선택하는 방식입니다. 다른 하나는 '화면에서 스포이드' 기능으로, 브라우저 창에 띄워진 어떤 웹사이트든 실시간으로 색상을 **웹 색상 추출**할 수 있습니다.

### 2단계: 원하는 지점 클릭
이미지가 로드되면 돋보기 모양의 스포이드 커서가 나타납니다. 마우스를 움직여 색상을 확인하고 싶은 픽셀 위에 정확히 클릭하세요. 커서 주변 영역이 확대되어 미세한 색상 차이까지 정밀하게 선택할 수 있습니다. 이 기능은 **이미지 색상 분석** 작업에서 특히 유용합니다.

### 3단계: 색상 코드 복사
클릭과 동시에 우측 패널에 해당 지점의 색상 정보가 표시됩니다. 기본값으로 HEX 코드가 보이며, 탭 하나만 누르면 RGB나 HSL 값으로 전환됩니다. '복사' 버튼을 누르면 클립보드에 저장되어, 포토샵, 피그마, VS Code 등 원하는 도구에 바로 붙여넣을 수 있습니다. 이렇게 간단한 **색상 코드 추출** 과정 덕분에 작업 시간이 획기적으로 줄어듭니다.

### 하위 팁: 색상 팔레트 저장하기
WooaColor의 또 다른 장점은 추출한 색상들을 모아 **팔레트 생성**이 가능하다는 점입니다. 여러 장의 이미지에서 포인트 컬러를 뽑아낸 후, '팔레트에 추가' 버튼을 누르면 리스트가 쌓입니다. 완성된 팔레트는 한 번에 내보내기(Export)하여 프로젝트 레퍼런스로 활용할 수 있습니다.
