# VS Code 줄 번호 표시 방법: 실무자도 놓치는 핵심 설정

코드 편집기를 사용하다 보면 가장 기본적이면서도 중요한 설정 중 하나가 **VS Code 줄 번호 표시**입니다. 디버깅, 코드 리뷰, 협업 시 줄 번호가 없으면 정확한 위치를 지칭하기 어렵고 생산성이 떨어집니다. 하지만 많은 개발자가 이 간단한 설정을 찾느라 시간을 낭비합니다. 이 글에서는 **VS Code 줄 번호 표시 방법**을 실무자 관점에서 압축하여 알려드립니다.

## VS Code 줄 번호 표시 방법: 3단계로 해결

아래 단계를 따라 하면 **VS Code 줄 번호 표시**를 30초 안에 설정할 수 있습니다. 별도 플러그인 설치 없이 기본 기능만으로 충분합니다.

### 1단계: 설정 창 열기
- VS Code를 실행한 후 좌측 하단 톱니바퀴 아이콘(설정)을 클릭하거나, 단축키 `Ctrl + ,`(Mac의 경우 `Cmd + ,`)를 누릅니다.

### 2단계: 검색창에 키워드 입력
- 열린 설정 화면 상단 검색창에 `line numbers`를 입력합니다. 또는 `editor line numbers`로 검색해도 동일한 결과가 나옵니다.

### 3단계: 값 변경 및 적용
- `Editor: Line Numbers` 항목을 찾아 드롭다운 메뉴를 클릭합니다. 기본값은 `on`이지만, 가끔 `off`나 `relative`로 설정된 경우가 있습니다. **on**으로 변경하면 즉시 에디터 왼쪽에 숫자가 표시됩니다.

설정이 완료되면 코드 블록마다 줄 번호가 나타나며, 디버깅 시 오류 위치를 바로 파악할 수 있습니다.
