## FontKit 주요 기능 상세

FontKit은 크롬 브라우저 확장 프로그램으로 동작하며, 방문한 웹페이지의 모든 폰트 정보를 실시간으로 분석해줍니다. 단순히 폰트 이름만 보여주는 것이 아니라, 실제 파일을 다운로드할 수 있도록 지원한다는 점이 핵심입니다.

### 웹폰트 추출과 저장 과정

**FontKit의 작동 방식**은 매우 직관적입니다.

1.  **확장 프로그램 설치**: 크롬 웹 스토어에서 FontKit을 무료로 설치합니다.
2.  **페이지 방문**: 원하는 폰트가 있는 웹사이트를 엽니다.
3.  **아이콘 클릭**: 브라우저 우측 상단의 FontKit 아이콘을 클릭하면, 현재 페이지에서 사용 중인 모든 웹폰트 목록이 나타납니다.
4.  **선택 및 다운로드**: 원하는 폰트를 선택하고 "Download" 버튼을 누르면 끝입니다. TTF, WOFF, WOFF2 등 여러 형식으로 저장할 수 있어 **한글 폰트 변환**이 필요 없습니다.

이 과정은 **30초**면 충분합니다. 별도의 설치 파일이나 회원 가입이 전혀 필요 없어 **무료 폰트 다운로드 도구** 중에서도 접근성이 가장 뛰어납니다.

### 실제 사용 후기: 장점과 단점을 솔직하게

실제로 디자인 작업을 하면서 FontKit을 사용해본 경험을 바탕으로 장단점을 정리해봤습니다.

**장점**

- **속도와 편의성**: 개발자 도구를 켜고 CSS를 뒤질 필요가 없습니다. 한 번의 클릭으로 즉시 폰트를 추출합니다.
- **보안**: "파일이 인터넷에 업로드되나요?"라는 의문이 들 수 있는데, FontKit은 브라우저 내에서 모든 작업을 처리합니다. 어떠한 데이터도 외부 서버로 전송되지 않아 안심하고 사용할 수 있습니다.
- **지원 형식 다양성**: 예쁜 한글 폰트를 발견했을 때, 원본 형식 그대로 저장할 수 있어 추가 변환 작업이 필요 없습니다.

**단점**

- **일부 폰트 미지원**: 모든 웹폰트가 추출되는 것은 아닙니다. 일부 사이트는 폰트 로딩 방식을 보호하거나, 폰트 파일에 접근 제한을 걸어둔 경우가 있습니다. 이럴 때는 FontKit이 해당 폰트를 인식하지 못합니다.
- **저작권 인식 필요**: 이 도구는 기술적으로 다운로드를 가능하게 해주지만, 모든 폰트가 무료로 사용 가능한 것은 아닙니다. 상업용 폰트를 무단으로 저장하는 것은 저작권 침해가 될 수 있으므로, **폰트 라이선스**를 반드시 확인해야 합니다.
- **Chrome 전용**: 현재는 크롬 브라우저에서만 작동하기 때문에, 사파리나 파이어폭스 사용자는 이용이 어렵습니다.
