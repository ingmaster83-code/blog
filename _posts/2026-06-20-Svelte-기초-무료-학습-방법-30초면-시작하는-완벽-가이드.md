Svelte를 배우고 싶은데 막막하다면, 당신만 그런 것이 아닙니다. 많은 입문자들이 "npm install"과 "vite" 같은 복잡한 개발 환경 설정 앞에서 좌절합니다. React나 Vue 같은 다른 프레임워크와 달리 Svelte는 문법이 직관적이지만, 첫 단계인 환경 구성 자체가 가장 큰 장벽으로 작용합니다. 결국 배우길 포기하고 마는 경우가 태반입니다.

**Svelte 기초 무료 학습 방법**의 핵심은 '환경 설정의 단순화'에 있습니다. 복잡한 설치 과정 없이, 순수하게 Svelte의 반응성(Reactivity)과 컴포넌트 개념 자체에 집중할 수 있는 도구가 필요합니다. 바로 [WooaDev](https://wooadev.wooahouse.com)가 그 해결책입니다. 이 도구는 온라인 코드 편집기이자 시각적 학습 플랫폼으로, 브라우저만 열면 즉시 Svelte 코드를 작성하고 결과를 실시간으로 확인할 수 있습니다. 별도의 터미널이나 패키지 매니저가 전혀 필요 없습니다.

## WooaDev로 Svelte 입문하는 단계별 가이드

### 1. 접속과 기본 화면 이해하기
[WooaDev](https://wooadev.wooahouse.com)에 접속하면 왼쪽에는 코드 편집기, 오른쪽에는 미리보기 화면이 나타납니다. 이 구조는 마치 공식 Svelte REPL과 유사하지만, 더 직관적인 UI로 설계되어 있습니다. 상단 메뉴에서 'Svelte' 템플릿을 선택하면 기본적인 Hello World 코드가 자동으로 로드됩니다.

### 2. 첫 번째 컴포넌트 만들기
`App.svelte` 파일에 `let name = 'Svelte';`라는 변수를 선언하고, `<h1>Hello {name}!</h1>`로 출력해보세요. 오른쪽 미리보기 화면이 즉시 업데이트됩니다. 이것이 Svelte의 반응성(Reactivity)입니다. 선언한 변수가 변경되면 DOM이 자동으로 업데이트되는 Svelte의 핵심 개념을 코드 몇 줄로 체험할 수 있습니다.

### 3. 이벤트 핸들링과 상태 관리 실습
```svelte
<script>
  let count = 0;
  function handleClick() {
    count += 1;
  }
</script>
<button on:click={handleClick}>클릭: {count}</button>
```
위 코드를 붙여넣고 버튼을 클릭해보세요. Svelte의 가장 강력한 기능 중 하나인 `on:click` 디렉티브와 상태 변화를 1초 만에 실습할 수 있습니다. 복잡한 `useState` 훅 없이도 직관적인 상태 관리를 경험할 수 있습니다.
