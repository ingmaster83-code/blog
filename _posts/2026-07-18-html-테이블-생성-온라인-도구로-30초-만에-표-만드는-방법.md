### 1단계: 도구 열기

웹 브라우저를 열고 WooaDev 사이트에 접속합니다. 별도의 다운로드나 로그인 과정이 없으므로 바로 편집 화면이 나타납니다.

### 2단계: 표 구조 설정

기본적으로 3x3 표가 제공됩니다. 필요에 따라 상단 메뉴에서 **행 추가**, **열 추가** 버튼을 클릭하거나, 마우스로 테두리를 드래그하여 크기를 조절할 수 있습니다. 최대 100개 이상의 행과 열을 자유롭게 구성할 수 있어 복잡한 데이터도 문제없습니다.

### 3단계: 데이터 입력

각 셀을 더블클릭하면 바로 편집 모드로 전환됩니다. 텍스트를 입력하거나 숫자, 날짜 등 어떤 형식의 데이터든 자유롭게 작성할 수 있습니다. 실시간으로 표가 업데이트되므로 입력 상태를 바로 확인할 수 있습니다.

### 4단계: 스타일 커스터마이징

헤더 색상, 테두리 두께, 셀 배경색, 글꼴 크기 등을 우측 패널에서 슬라이더와 드롭다운으로 조정합니다. 복잡한 CSS를 몰라도 **테이블 스타일링**이 가능합니다. 예를 들어 헤더 행을 파란색으로 강조하거나, 교차 행에 다른 색상을 적용하는 것도 클릭 한 번입니다.

### 5단계: 코드 내보내기

표 편집이 끝나면 하단의 **HTML 복사** 버튼을 클릭합니다. 생성된 코드는 `<table>` 태그부터 인라인 CSS까지 완벽하게 포함되어 있습니다. 이 코드를 워드프레스, 티스토리, 또는 자체 제작한 웹페이지의 HTML 편집기에 붙여넣으면 바로 적용됩니다.

## 주의사항: 온라인 도구 사용 시 체크리스트

아무리 편리한 **html 테이블 생성 온라인 도구**라도 몇 가지 주의할 점이 있습니다.

- **반응형 확인**: 생성된 표는 기본적으로 고정 너비일 수 있습니다. 모바일에서 볼 경우 좌우 스크롤이 생길 수 있으므로, CSS에 `max-width: 100%`와 `overflow-x: auto`를 추가하는 것을 권장합니다.
- **데이터 백업**: 온라인 도구는 브라우저 로컬 저장소에 데이터를 임시 저장하지만, 브라우저 기록을 지우면 사라집니다. 중요한 표는 완성 후 반드시 HTML 코드를 별도 파일로 저장하세요.
