## WooaColor 사용법 단계별 가이드

### 1. 기본 색상 선택하기

WooaColor에 접속하면 중앙에 큰 컬러 피커가 나타납니다. 원하는 기본 색상을 클릭하거나 HEX 코드를 직접 입력하면 됩니다. 예를 들어 브랜드 메인 컬러인 #2A6BFF를 입력했더니, 도구가 자동으로 어울리는 보조 색상들을 추천해줬습니다.

### 2. CSS 변수 자동 생성 확인

색상을 고르면 오른쪽 패널에 CSS 변수 코드가 실시간으로 나타납니다. `--primary: #2A6BFF; --secondary: #4A90D9; --accent: #FF6B6B;` 이런 식으로 -- 접두어를 가진 변수들이 자동 정리됩니다. CSS 변수 생성기가 이렇게 빠를 줄 몰랐습니다.

### 3. 팔레트 저장 및 복사

마음에 드는 무료 컬러 팔레트가 완성되면 오른쪽 상단의 "복사" 버튼을 클릭하세요. 클립보드에 CSS 코드가 바로 저장됩니다. 그다음 프로젝트의 `:root` 영역에 붙여넣기만 하면 끝입니다. 손수 타이핑할 필요가 전혀 없어요.

### 4. 변수 값 수정하기

만약 색상 톤을 조금 바꾸고 싶다면, 각 색상 옆에 있는 슬라이더로 명도나 채도를 조절할 수 있습니다. 변경할 때마다 CSS 코드가 자동 업데이트되므로, 실시간으로 결과를 확인하면서 색상 변수를 다듬을 수 있습니다.
