### 1단계: 색상 팔레트 생성하기

WooaColor 사이트에 접속하면 직관적인 컬러 피커가 나타납니다. 원하는 색상을 클릭하거나 HEX 값을 직접 입력하여 팔레트를 구성할 수 있습니다. 색상 추출 기능을 활용해 기존 이미지에서 색상을 가져오는 것도 가능합니다. 최대 10개의 색상을 한 번에 관리할 수 있어 소규모 프로젝트에 적합합니다.

각 색상 옆에는 자동으로 HEX, RGB, HSL 값이 표시됩니다. 이 정보는 실시간으로 업데이트되며, 팔레트를 수정할 때마다 즉시 반영됩니다. 개발자가 요구하는 특정 형식이 있다면 미리 확인하고 조정할 수 있습니다.

팔레트가 완성되면 화면 우측 하단의 '내보내기' 버튼을 클릭합니다. 드롭다운 메뉴에서 'JSON'을 선택하면 자동으로 JSON 파일이 생성됩니다. 이 파일에는 모든 색상 코드가 객체 배열 형태로 정리되어 있어 복사해서 바로 사용하거나 다운로드할 수 있습니다. **색상 코드 JSON 내보내기** 과정이 이렇게 간단합니다.

### 4단계: 내보낸 데이터 활용하기

생성된 JSON 데이터는 다음과 같은 구조로 되어 있습니다:
```json

```
