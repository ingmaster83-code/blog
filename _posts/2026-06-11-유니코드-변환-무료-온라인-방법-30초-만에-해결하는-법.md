### 주요 기능: 간편한 유니코드 디코더와 인코더

WooaText는 단순히 유니코드 변환만 해주는 게 아닙니다. **문자 인코딩**과 **디코딩**을 모두 지원합니다. 구체적으로 어떤 기능이 있는지 살펴볼까요?

#### 1. URL 디코딩과 인코딩
웹 브라우저 주소창에 보이는 `%ED%95%9C%EA%B8%80` 같은 퍼센트 인코딩 값을 한글로 바로 바꿔줍니다. 반대로 한글을 입력하면 URL 형식으로 자동 변환됩니다. 웹 데이터를 다루는 개발자나 마케터에게 특히 유용합니다.

#### 2. 유니코드 이스케이프 시퀀스 변환
`\uD55C\uAE00` 같은 자바스크립트 유니코드 문자열을 실제 문자로 바꾸거나, 반대로 변환할 수 있습니다. JSON 데이터를 분석할 때 큰 도움이 됩니다.

#### 3. 한글 깨짐 복구
가장 자주 쓰는 기능입니다. EUC-KR로 인코딩된 문서를 UTF-8로 잘못 열었을 때 발생하는 깨짐 현상을 몇 초 만에 복원합니다. 과제 자료가 깨졌을 때 이 기능 덕분에 여러 번 살았습니다.

#### 4. HTML 엔티티 변환
`&#54620;&#44544;` 같은 HTML 특수문자 코드를 일반 텍스트로 변환합니다. 웹페이지 소스를 복사했는데 이상한 문자가 섞여 있다면 이 기능으로 정리하세요.
