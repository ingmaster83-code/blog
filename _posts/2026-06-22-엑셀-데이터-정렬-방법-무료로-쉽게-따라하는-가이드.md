## 엑셀 데이터 정렬 방법: 기본부터 이해하기

엑셀에서 데이터를 정렬하는 것은 생각보다 간단합니다. 가장 기본적인 방법은 원하는 열을 선택한 뒤, 오름차순 또는 내림차순 버튼을 클릭하는 것입니다. 하지만 데이터가 많거나 조건이 복잡할 때는 조금 더 세심한 접근이 필요합니다. 여기서는 텍스트 정렬과 숫자 정렬을 각각 다루며, **데이터 정렬**의 핵심을 잡아보겠습니다.

### 기본 정렬: 한 번의 클릭으로 끝내기

1. 정렬할 데이터가 있는 표 전체를 드래그하여 선택합니다.
2. 엑셀 상단 메뉴에서 '데이터' 탭을 클릭합니다.
3. '오름차순 정렬' (A→Z) 또는 '내림차순 정렬' (Z→A) 아이콘을 누르면 즉시 정렬됩니다.

이 방법은 한 가지 기준으로만 정렬할 때 유용합니다. 예를 들어 고객 이름을 가나다 순으로 정리하거나, 판매 금액을 높은 순서대로 보고 싶을 때 사용하세요.

### 다중 조건 정렬: 여러 열을 동시에 정리하기

실무에서는 하나의 열만으로 정렬이 부족할 때가 많습니다. 예를 들어, 부서별로 먼저 정리한 뒤 같은 부서 안에서 연봉 순으로 정렬해야 한다면 다중 조건 정렬이 필요합니다.

1. 데이터 범위를 선택한 후, '데이터' 탭에서 '정렬' 버튼을 클릭합니다.
2. 첫 번째 기준 열을 '부서'로 설정하고, 정렬 순서를 '오름차순'으로 지정합니다.
3. '수준 추가' 버튼을 눌러 두 번째 기준 열을 '연봉'으로 설정하고, '내림차순'을 선택합니다.
4. 확인을 누르면 부서별로 묶인 후, 연봉이 높은 순서대로 데이터가 정리됩니다.

이처럼 **엑셀 오름차순 정렬**과 **엑셀 내림차순 정렬**을 조합하면 복잡한 데이터도 깔끔하게 정리할 수 있습니다.
