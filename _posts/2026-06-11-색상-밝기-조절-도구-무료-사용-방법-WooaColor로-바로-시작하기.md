## 주요 기능: HSL 조절로 완벽한 색상 만들기

WooaColor는 크게 세 가지 핵심 기능을 제공합니다. 각 기능을 초보자도 쉽게 이해할 수 있도록 단계별로 설명드리겠습니다.

### 1. 색상 선택: 휠과 코드 입력 두 가지 방법

가장 먼저 할 일은 편집할 기본 색상을 고르는 것입니다.

1. **컬러 휠 사용**: 마우스로 원형 팔레트에서 원하는 색상을 클릭하면 즉시 해당 색상이 적용됩니다.
2. **코드 직접 입력**: 이미 HEX 코드(#FF5733 등)나 RGB 값(255, 87, 51)을 알고 있다면 입력창에 바로 입력하면 됩니다.
3. **실시간 미리보기**: 선택한 색상이 큰 미리보기 영역에 즉시 표시되므로 결과를 바로 확인할 수 있습니다.

### 2. 밝기 조절: HSL 슬라이더 활용법

이 도구의 핵심인 **색상 밝기 조절** 기능입니다.

- **L(Lightness) 슬라이더**: 밝기를 직접 조절합니다. 0%로 내리면 검은색, 50%면 원래 색상, 100%면 흰색이 됩니다.
- **H(Hue) 슬라이더**: 색조를 0~360도 범위에서 변경합니다. 같은 밝기와 채도에서 다른 계열의 색으로 바꿀 수 있습니다.
- **S(Saturation) 슬라이더**: 채도를 조절합니다. 0%면 회색톤, 100%면 가장 선명한 색이 됩니다.

**팁**: 밝기만 조절하고 싶다면 H와 S 슬라이더는 건드리지 말고 L 슬라이더만 움직이면 됩니다. 이렇게 하면 원래 색감을 유지하면서 밝기만 자연스럽게 변화시킬 수 있습니다.

### 3. 코드 확인 및 복사: 디자인 작업에 바로 활용

조절이 끝난 색상은 즉시 실용적인 형태로 변환됩니다.

- **HEX 코드**: 웹 디자인에 가장 많이 사용되는 16진수 코드입니다. 복사 버튼을 클릭하면 클립보드에 저장됩니다.
- **RGB 코드**: 그래픽 디자인 툴이나 프린팅 작업에 유용한 RGB 값입니다.
- **HSL 코드**: CSS 작업을 하는 경우 직접 HSL 값을 사용할 수 있어 편리합니다.

이렇게 복사한 코드를 포토샵, 일러스트레이터, 피그마, 또는 HTML/CSS 코드에 붙여넣기만 하면 됩니다.
