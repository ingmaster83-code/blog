## WooaQR 사용법: QR코드 CSV 대량 생성 단계별 가이드

WooaQR을 사용하여 QR코드를 대량 생성하는 과정은 매우 직관적입니다. 아래 단계를 따라해보세요.

### 1단계: CSV 파일 준비하기

먼저 생성할 QR코드의 데이터가 담긴 CSV 파일을 준비합니다. 첫 번째 열에 URL 또는 텍스트를 입력하고, 두 번째 열 이후에는 파일명이나 식별자 등 추가 정보를 넣을 수 있습니다.

**CSV 파일 예시:**
```
URL,상품명
[https://example.com/product1,제품A](https://example.com/product1,제품A)
[https://example.com/product2,제품B](https://example.com/product2,제품B)
[https://example.com/product3,제품C](https://example.com/product3,제품C)
```

**꿀팁:** 엑셀 또는 구글 스프레드시트에서 데이터를 정리한 후 CSV(쉼표로 구분) 형식으로 내보내면 편리합니다.

### 2단계: WooaQR 접속 및 파일 불러오기

웹 브라우저에서 [WooaQR](https://qrkit.wooahouse.com)에 접속합니다. 별도의 회원가입이나 로그인은 필요 없습니다. 화면에 보이는 "CSV 파일 선택" 버튼을 클릭하여 준비한 CSV 파일을 불러옵니다.

**비교 포인트:** 다른 도구는 회원가입 후 로그인해야 사용할 수 있는 경우가 많지만, WooaQR은 즉시 사용 가능합니다. 이는 긴급한 작업이 필요할 때 큰 장점입니다.

### 3단계: QR코드 옵션 설정하기

파일을 불러오면 자동으로 데이터를 분석하여 QR코드 미리보기를 보여줍니다. 이때 다음과 같은 옵션을 설정할 수 있습니다.

- **QR코드 크기:** 기본값(200x200)부터 최대 1000x1000까지 조절 가능
- **색상:** 검정 외에도 다양한 색상 선택 가능 (브랜드 색상 매칭에 유용)
- **로고 삽입:** 중앙에 회사 로고나 아이콘 추가 가능

옵션 설정은 선택 사항이며, 기본 설정으로도 충분히 깔끔한 QR코드를 생성할 수 있습니다.

### 4단계: QR코드 생성 및 다운로드

"생성" 버튼을 클릭하면 모든 CSV 데이터에 대한 QR코드가 즉시 생성됩니다. 생성된 QR코드는 ZIP 파일 형태로 한 번에 다운로드할 수 있습니다. 각 이미지 파일명은 CSV 파일의 두 번째 열 데이터나 행 번호로 자동 지정됩니다.

**효율성 비교:** 타사 도구는 하나씩 다운로드해야 하는 경우가 많지만, WooaQR은 일괄 다운로드로 시간을 크게 절약해줍니다.

### 5단계: QR코드 활용하기

다운로드한 QR코드는 다음과 같은 용도로 바로 활용할 수 있습니다.

- 제품 포장에 부착
- 이벤트 초대장에 인쇄
- 출입 명부에 활용
- 재고 관리 라벨로 사용

**응용 팁:** QR코드 이미지 파일명을 CSV의 데이터와 일치시키면 관리가 훨씬 수월합니다. 예를 들어 상품코드를 파일명으로 지정하면 나중에 매칭하기 쉽습니다.

## QR코드 CSV 대량 생성 시 주의사항

WooaQR을 사용할 때 몇 가지 주의할 점이 있습니다.

**데이터 형식 확인:** CSV 파일의 첫 번째 열은 반드시 QR코드로 변환할 데이터(URL 또는 텍스트)여야 합니다. 한글이 포함된 경우 인코딩 문제가 발생할 수 있으므로 UTF-8 형식으로 저장하는 것이 좋습니다.

**브라우저 호환성:** Chrome, Edge, Firefox 등 최신 브라우저에서 최적의 성능을 발휘합니다. Internet Explorer는 지원되지 않으니 주의하세요.

**생성 속도:** 수천 개의 QR코드를 한 번에 생성할 경우 브라우저 메모리 사용량이 증가할 수 있습니다. 500개 이상의 데이터를 처리할 때는 브라우저가 응답하지 않는 현상을 방지하기 위해 파일을 분할하여 처리하는 것을 권장합니다.

**QR코드 테스트:** 대량 생성 후 반드시 샘플 몇 개를 스마트폰으로 스캔하여 정상 작동하는지 확인하세요. URL 오류나 깨짐 현상을 사전에 방지할 수 있습니다.
