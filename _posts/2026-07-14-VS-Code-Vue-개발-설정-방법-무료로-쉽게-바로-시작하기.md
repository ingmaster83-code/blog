### WooaVS가 제공하는 핵심 기능 3가지

#### 1. 지능형 코드 자동 완성과 스니펫
WooaVS는 Vue 템플릿 구문, `v-for`, `v-if` 디렉티브, Composition API의 `ref()`, `reactive()` 등을 타이핑할 때 실시간으로 추천 목록을 보여줍니다. 또한 `vbase`를 입력하면 Vue Single File Component의 기본 구조가 자동 생성되어 템플릿 작업 시간을 50% 이상 단축할 수 있습니다.

#### 2. 실시간 오류 감지와 포맷팅
코드를 작성하는 순간 ESLint와 Prettier가 통합 작동하여 문법 오류와 스타일 문제를 하이라이트로 표시합니다. 저장(Ctrl+S)만 하면 자동으로 코드가 정리되므로, 별도로 포맷팅 명령어를 실행할 필요가 없습니다. 이 기능은 Vue.js의 단방향 데이터 흐름을 유지하는 데 큰 도움을 줍니다.

#### 3. vue-router와 Vuex/ Pinia 지원
