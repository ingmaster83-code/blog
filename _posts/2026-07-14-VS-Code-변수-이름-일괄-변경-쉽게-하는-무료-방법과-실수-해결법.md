## WooaVS로 안전하게 변수명 변경하는 방법

### 웹 기반이라 설치 불필요

WooaVS의 가장 큰 장점은 웹 브라우저에서 바로 실행된다는 점입니다. VS Code 플러그인을 설치할 필요도, 복잡한 환경 설정을 할 필요도 없습니다. 그냥 사이트에 접속해서 코드를 붙여넣기만 하면 됩니다. 덕분에 회사 보안 정책 때문에 플러그인 설치가 제한된 환경에서도 부담 없이 사용할 수 있습니다.

### 변경 전 미리보기로 실수 방지

이 도구의 핵심 기능은 **변경 전 미리보기**입니다. `from`과 `to` 필드에 바꿀 변수명을 입력하면, 코드 내에서 일치하는 모든 부분이 하이라이트되어 보여집니다. 이 단계에서 사용자는 "이 부분은 진짜 바꿔도 되는가?"를 하나씩 눈으로 확인할 수 있습니다. 저처럼 전체 바꾸기를 눌렀다가 `database`가 깨지는 실수를 근본적으로 막아주는 겁니다.

### 정규식과 고급 필터로 정밀 제어

단순 문자열 치환을 넘어, 정규식(Regex)을 지원하므로 `temp_`로 시작하는 변수만 골라내거나, 특정 패턴의 변수명만 선택적으로 변경할 수 있습니다. 예를 들어 `tempUserId`, `tempUserName` 같은 변수들을 `tmpUserId`, `tmpUserName`으로 한 번에 변경할 때 매우 유용합니다.
