## WooaColor 사용법 단계별 가이드

### 1. WooaColor 웹사이트 접속

먼저 브라우저를 열고 [WooaColor](https://colorkit.wooahouse.com)에 접속합니다. 별도의 설치나 가입 없이 즉시 사용할 수 있습니다.

### 2. 텍스트 그림자 탭 선택

페이지 상단 메뉴에서 "텍스트 그림자" 또는 "Text Shadow" 탭을 클릭합니다. 메인 화면에 있는 텍스트 입력창과 슬라이더 컨트롤이 바로 표시됩니다.

### 3. 기본 텍스트 입력

"미리보기 텍스트" 입력란에 원하는 문구를 자유롭게 입력합니다. 기본값으로 "Hello World"가 설정되어 있으나, 실제 적용할 텍스트로 변경하면 더 정확한 미리보기가 가능합니다.

### 4. 그림자 속성 조정하기

**CSS text-shadow 생성**에 필요한 주요 속성을 슬라이더로 조절합니다.

- **Offset X**: 그림자의 가로 위치를 조정합니다. 양수는 오른쪽, 음수는 왼쪽으로 이동합니다.
- **Offset Y**: 그림자의 세로 위치를 조정합니다. 양수는 아래쪽, 음수는 위쪽으로 이동합니다.
- **Blur**: 그림자의 흐림 정도를 설정합니다. 값이 클수록 부드럽고 퍼지는 효과가 나타납니다.
- **Color**: 색상 피커를 통해 원하는 그림자 색상을 선택합니다. 투명도도 함께 조절할 수 있습니다.

각 값을 변경할 때마다 우측의 미리보기 영역이 실시간으로 업데이트되므로 직관적으로 디자인을 완성할 수 있습니다.

### 5. 다중 그림자 추가 (심화 기능)

하나의 텍스트에 여러 개의 그림자를 겹치고 싶다면 "+" 버튼을 클릭해 레이어를 추가합니다. 각 레이어마다 독립적으로 X, Y, 블러, 색상을 설정할 수 있어 네온사인, 3D 입체 효과 등 고급 **텍스트 효과**를 자유롭게 구현할 수 있습니다.

### 6. CSS 코드 복사하기

원하는 스타일이 완성되면 화면 하단에 표시된 CSS 코드를 확인합니다. `text-shadow: 2px 2px 4px rgba(0,0,0,0.5);` 같은 형식의 코드가 자동 생성됩니다. "복사" 버튼을 클릭하면 클립보드에 저장되며, 자신의 프로젝트 CSS 파일에 붙여넣기만 하면 됩니다.

## 사용 시 주의사항

- **브라우저 호환성 확인**: 모든 최신 브라우저는 `text-shadow` 속성을 지원하지만, 구형 Internet Explorer(IE 9 이하)에서는 정상 표시되지 않을 수 있습니다. 필요한 경우 Fallback 스타일을 추가하는 것을 권장합니다.
- **과도한 블러 지양**: 블러 값을 너무 높이면 텍스트의 가독성이 크게 떨어집니다. 일반적으로 2~8px 범위 내에서 조정하는 것이 좋습니다.
- **색상 대비 확인**: 그림자 색상이 배경색과 너무 유사하면 효과가 거의 보이지 않습니다. 항상 배경 대비를 고려해 색상을 선택하세요.
- **모바일 최적화**: 모바일 환경에서도 그림자가 잘 보이는지 확인해야 합니다. 작은 화면에서는 그림자 두께를 약간 줄이는 것이 좋습니다.
- **다중 그림자 성능**: 그림자 레이어를 5개 이상 사용하면 렌더링 성능에 영향을 줄 수 있습니다. 특히 애니메이션과 함께 사용할 때는 주의가 필요합니다.
