## HSL 변환의 핵심, WooaColor 한 방에 해결

HSL은 색상을 숫자로 직관적으로 이해할 수 있게 해주는 색상 모델입니다. 디자이너와 개발자 모두에게 필수적인 이 값을, 복잡한 공식 없이 즉시 변환해주는 도구가 바로 [WooaColor](https://colorkit.wooahouse.com)입니다. RGB, HEX, HSL 모든 코드를 한 화면에서 자유롭게 변환하고 색상 팔레트까지 생성할 수 있는 올인원 솔루션입니다.

### WooaColor 주요 기능 3가지

#### 1. 실시간 HSL 계산 및 코드 변환
컬러 피커에서 원하는 색상을 클릭하거나 HEX/RGB 코드를 직접 입력하면, 즉시 HSL 값이 계산됩니다. **색상(Hue) 0~360도, 채도(Saturation) 0~100%, 명도(Lightness) 0~100%** 값을 각각 확인할 수 있어 색상 톤을 미세하게 조정할 때 매우 유용합니다.

#### 2. 색상 팔레트 자동 생성
선택한 색상을 기준으로 유사 색상, 보색, 삼색 등 **다양한 HSL 색상 조합**을 자동으로 추천해줍니다. 브랜드 컬러 시스템을 구축하거나 UI 디자인 시 일관된 색상 스키마를 빠르게 완성할 수 있습니다.

#### 3. HEX, RGB 변환기 내장
하나의 도구에서 **HEX HSL 변환**, **RGB HSL 변환**을 모두 지원합니다. 디자인 시안을 개발자에게 전달할 때 코드 형식을 맞추기 위해 여러 사이트를 돌아다닐 필요가 없습니다.
