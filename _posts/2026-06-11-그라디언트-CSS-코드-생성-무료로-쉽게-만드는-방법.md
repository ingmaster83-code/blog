## WooaColor 사용법 단계별 가이드

### 1. 웹사이트 접속하기

먼저 [WooaColor](https://colorkit.wooahouse.com)에 접속합니다. 메인 화면에 두 개의 색상 선택 창과 실시간 미리보기 화면이 보입니다. 별도의 다운로드나 설치 과정이 전혀 없습니다.

### 2. 시작 색상과 끝 색상 선택하기

좌측에 있는 첫 번째 색상 아이콘을 클릭하면 색상 팔레트가 열립니다. 원하는 시작 색상을 선택하거나, 하단의 HEX 코드 입력창에 직접 색상 코드를 입력할 수 있습니다. 동일한 방법으로 우측의 끝 색상도 선택합니다.

예를 들어 파란색 계열의 그라데이션을 만들고 싶다면 시작 색상에 `#667eea`, 끝 색상에 `#764ba2`를 입력해보세요. 우아한 보라-파랑 그라디언트가 즉시 생성됩니다.

### 3. 그라디언트 방향 설정하기

색상 선택 아래에 있는 방향 조절 슬라이더를 움직이면 그라디언트의 진행 각도를 변경할 수 있습니다. `0deg`는 위에서 아래로, `90deg`는 왼쪽에서 오른쪽으로 색상이 전환됩니다. 원하는 느낌이 나올 때까지 슬라이더를 조정해보세요.

### 4. CSS 코드 복사하기

원하는 그라데이션이 완성되면 화면 하단에 실시간으로 업데이트되는 CSS 코드를 확인할 수 있습니다. 코드 옆에 있는 복사 아이콘을 클릭하면 클립보드에 자동으로 복사됩니다.

복사된 코드는 다음과 같은 형식입니다:
```css

```
이 코드를 웹사이트 CSS 파일이나 HTML 스타일 속성에 붙여넣기만 하면 됩니다.

### 5. 추가 색상으로 고급 설정하기

두 가지 색상 이상의 복잡한 그라디언트가 필요하다면, 색상 선택 영역 하단의 + 버튼을 클릭해 세 번째, 네 번째 색상을 추가할 수 있습니다. 각 색상의 위치(퍼센트)도 드래그로 조절 가능합니다.
