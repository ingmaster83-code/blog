## 주요 기능: 텍스트 N번째 문자 삭제를 포함한 5가지 핵심 기능

### 1. N번째 문자 삭제 기능 사용법

이 도구의 핵심 기능은 텍스트의 원하는 위치에 있는 글자를 정확히 제거하는 것입니다.

**단계별 사용법:**
1. [WooaText](https://textkit.wooahouse.com) 페이지에 접속합니다.
2. 왼쪽 입력 창에 삭제할 텍스트를 붙여넣습니다.
3. "N번째 문자 삭제" 옵션에서 지우고 싶은 위치 번호를 입력합니다 (예: 1을 입력하면 첫 글자 삭제).
4. "실행" 버튼을 누르면 오른쪽 출력 창에 결과가 즉시 표시됩니다.

### 2. 문자열 자르기와 특정 위치 문자 제거

텍스트 N번째 문자 삭제 외에도 다양한 **문자열 자르기** 기능을 제공합니다.
- 처음 N개 문자만 남기기
- 마지막 N개 문자 삭제
- 특정 구분자로 분할 후 원하는 부분 추출

이 기능들은 특히 주소록에서 앞자리 전화번호만 추출하거나, 제품 코드에서 불필요한 접두사를 제거할 때 자주 사용됩니다.

### 3. 줄 단위 일괄 처리

여러 줄의 텍스트가 있다면 모든 줄에 동일한 처리를 한 번에 적용할 수 있습니다. 예를 들어 100개의 이메일 주소에서 @ 앞부분만 추출할 때 매우 효과적입니다. 이렇게 **온라인 텍스트 편집기**의 장점을 최대한 활용할 수 있습니다.

### 4. 공백 및 특수문자 정리

데이터를 복사해 오면 불필요한 공백이나 탭 문자가 섞여 있는 경우가 많습니다. 이 도구는 연속된 공백 제거, 줄바꿈 정리, 특수문자 일괄 삭제 기능도 제공합니다.
