### WooaColor 사용법 단계별 가이드

#### 1. 기본 팔레트 만들기

사이트에 접속하면 기본 5가지 색상 팔레트가 보입니다. 각 색상 원을 클릭하면 컬러 피커가 열리고, 원하는 색으로 자유롭게 변경할 수 있습니다. 색상 코드(HEX, RGB 등)를 직접 입력해도 됩니다.

#### 2. 색상 추가 및 삭제

팔레트 하단의 "+" 버튼을 클릭하면 색상을 최대 10개까지 추가할 수 있습니다. 불필요한 색상은 해당 원의 우측 상단 "X" 버튼으로 삭제하면 됩니다. 이렇게 자유롭게 편집하면서 자신만의 컬러 팔레트를 완성할 수 있습니다.

#### 3. 팔레트 내보내기

완성된 팔레트는 다양한 방식으로 내보낼 수 있습니다. 우측 하단의 내보내기 버튼을 클릭하면 PNG 이미지, CSS 변수, SCSS, Tailwind CSS 등 여러 포맷이 나타납니다. 원하는 형식을 선택하면 클립보드에 복사되거나 이미지로 다운로드됩니다. 이 과정이 바로 **색상 팔레트 저장 내보내기 무료**의 핵심 기능입니다.

#### 4. 프로젝트에 바로 적용하기

내보낸 CSS 코드는 그대로 웹 프로젝트에 붙여넣을 수 있고, PNG 이미지는 PPT나 리포트에 첨부하기 좋습니다. 특히 Tailwind CSS 사용자라면 config 파일에 바로 복사하여 사용할 수 있어 매우 편리합니다.
