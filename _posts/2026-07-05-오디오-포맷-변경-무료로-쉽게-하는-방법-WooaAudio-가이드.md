## 오디오 포맷 변환 방법 단계별 가이드

이제 실제로 WooaAudio를 사용해 **오디오 변환**을 진행해보겠습니다. 아래 단계를 그대로 따라 하면 누구나 쉽게 변환할 수 있습니다.

### 1단계: WooaAudio 웹사이트 접속하기

먼저 [WooaAudio](https://wooaaudio.wooahouse.com)에 접속합니다. 사이트에 들어가면 깔끔한 화면이 나타나는데, 가운데에 파일을 업로드할 수 있는 영역이 보입니다. 특별한 버튼을 찾을 필요 없이 직관적으로 설계되어 있어 초보자도 당황하지 않습니다.

### 2단계: 변환할 오디오 파일 업로드하기

화면 중앙의 '파일 선택' 버튼을 클릭하거나, 파일을 드래그앤드롭으로 업로드 영역에 끌어다 놓습니다. 지원하는 파일 형식은 MP3, WAV, OGG, FLAC 등이며, 일반적인 오디오 파일이라면 대부분 문제없이 업로드됩니다.

**팁**: 여러 개의 파일을 한 번에 선택하여 업로드할 수 있습니다. Ctrl 키(맥은 Cmd 키)를 누른 상태에서 파일을 클릭하면 다중 선택이 가능합니다.

### 3단계: 원하는 출력 포맷 선택하기

파일이 업로드되면 자동으로 포맷 선택 창이 나타납니다. 여기서 원하는 출력 형식을 고르면 됩니다. 예를 들어:

- **MP3**: 가장 보편적인 포맷으로, 용량과 음질의 균형이 좋습니다.
- **WAV**: 무손실 고음질이 필요할 때 적합합니다.
- **OGG**: 오픈소스 포맷으로, 온라인 스트리밍에 자주 사용됩니다.
- **FLAC**: 음질을 최우선으로 하는 무손실 압축 포맷입니다.

원하는 포맷을 선택한 후 '변환' 버튼을 클릭하면 잠시 기다리면 됩니다.

### 4단계: 변환 완료 후 다운로드하기

변환이 완료되면 자동으로 다운로드가 시작됩니다. 만약 다운로드가 자동으로 시작되지 않으면 화면에 나타난 '다운로드' 버튼을 클릭하면 됩니다. 변환된 파일은 원본 파일명 그대로 저장되며, 확장자만 변경됩니다.
