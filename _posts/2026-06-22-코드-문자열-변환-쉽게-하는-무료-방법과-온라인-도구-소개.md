## WooaDev의 주요 기능과 사용 방법

이 도구가 제공하는 다양한 변환 기능을 하나씩 살펴보겠습니다. 각 기능은 모두 코드 문자열 변환 쉽게 처리할 수 있도록 설계되었습니다.

### URL 인코딩과 디코딩 기능

웹 개발에서 가장 자주 사용되는 기능 중 하나가 URL 인코딩입니다. 한글이나 특수 문자가 포함된 문자열을 URL 형식에 맞게 변환할 때 유용합니다.

1. [WooaDev](https://wooadev.wooahouse.com)에 접속합니다.
2. 입력창에 변환하고 싶은 문자열을 붙여넣습니다.
3. URL 인코딩 또는 디코딩 버튼을 클릭하면 즉시 결과가 출력됩니다.

예를 들어 "안녕하세요"라는 문자열을 URL 인코딩하면 "%EC%95%88%EB%85%95%ED%95%98%EC%84%B8%EC%9A%94"로 변환됩니다. 이 기능은 검색 파라미터를 처리하거나 API 요청을 보낼 때 필수적입니다.

### Base64 변환 기능

Base64는 바이너리 데이터를 텍스트 형식으로 안전하게 전송하기 위해 사용되는 인코딩 방식입니다. 이미지를 HTML에 직접 포함시키거나, 간단한 데이터 암호화가 필요할 때 자주 활용됩니다.

WooaDev의 Base64 변환기는 입력한 문자열을 즉시 인코딩하거나 디코딩해줍니다. 예를 들어 "Hello World"를 Base64로 인코딩하면 "SGVsbG8gV29ybGQ="가 됩니다. 이 기능을 통해 문자열 변환 과정을 훨씬 수월하게 진행할 수 있습니다.

### HTML 엔티티와 이스케이프 처리

HTML 문서 안에 특수 문자(<, >, & 등)를 그대로 사용하면 코드가 깨지거나 의도치 않은 동작이 발생할 수 있습니다. 이때 필요한 것이 HTML 엔티티 변환입니다.

WooaDev는 `<script>` 같은 태그를 `&lt;script&gt;`로 자동 변환해주며, 반대로 변환된 문자열을 원래대로 복원하는 디코딩도 지원합니다. 이 기능은 블로그 포스팅이나 코드 예제를 작성할 때 특히 유용합니다.
