### PDF 1페이지만 추출하는 세부 단계

#### 1. WooaPDF 사이트 접속하기

먼저 인터넷 브라우저를 열고 [WooaPDF 무료로 사용하기](https://pdfkit.wooahouse.com) 링크를 클릭하세요. 사이트에 들어가면 깔끔한 메인 화면이 나타납니다. 별도의 다운로드나 설치 과정이 전혀 없습니다.

#### 2. 파일 업로드하기

화면 중앙에 있는 파란색 '파일 선택' 버튼을 클릭한 후, 추출하고 싶은 PDF 파일을 선택하세요. 또는 컴퓨터 바탕화면에 있는 PDF 파일을 마우스로 끌어다가 사이트 위에 놓아도 바로 업로드됩니다. 파일 크기 제한이 거의 없어 대용량 문서도 문제없이 처리할 수 있습니다.

#### 3. 페이지 분할 옵션 선택하기

파일이 업로드되면 '페이지 분할' 메뉴가 자동으로 활성화됩니다. 여기서 두 가지 방식을 선택할 수 있습니다.

- **개별 페이지 추출**: 모든 페이지를 각각의 PDF 파일로 나누기
- **특정 페이지 추출**: 원하는 페이지만 골라서 하나의 PDF로 만들기

**PDF 1페이지만 추출**하려면 '특정 페이지 추출'을 선택하고, 텍스트 입력창에 해당 페이지 번호를 입력하세요. 예를 들어 5페이지 한 장만 뽑고 싶다면 숫자 '5'를 입력하면 됩니다. 여러 페이지를 연속으로 추출할 때는 '1-3'처럼 범위를 입력할 수도 있습니다.

#### 4. 변환 및 다운로드

페이지 번호 입력을 마쳤다면 초록색 '변환' 버튼을 누르세요. 잠시 기다리면(보통 1~2초) 화면 우측에 '다운로드' 버튼이 나타납니다. 이 버튼을 클릭하면 추출한 PDF 파일이 자동으로 컴퓨터에 저장됩니다. 단 한 번의 클릭으로 원하는 페이지만 깔끔하게 분리된 것을 확인할 수 있습니다.
