### 1단계: 기준 색상 선택하기

WooaColor 메인 화면에서 첫 번째 색상과 두 번째 색상을 선택합니다. 색상 피커를 클릭하거나 HEX 코드를 직접 입력할 수 있습니다. 브랜드 컬러나 기존 프로젝트에서 사용 중인 색상 코드를 붙여넣으면 바로 적용됩니다.

### 2단계: 혼합 비율 조정하기

슬라이더를 좌우로 드래그하여 각 색상의 혼합 비율을 조정합니다. 예를 들어 빨간색 70%와 파란색 30%로 설정하면 보라색 톤이 실시간으로 화면에 표시됩니다. **색상 혼합 비율 계산**이 즉시 이루어져 원하는 색감을 빠르게 찾을 수 있습니다.

### 3단계: 결과 복사해서 사용하기

원하는 혼합 색상이 완성되면 화면에 표시된 HEX 코드를 복사합니다. 이 코드를 디자인 툴이나 CSS 코드에 붙여넣으면 바로 사용할 수 있습니다. **웹디자인 색상 도구**로서 WooaColor는 별도 저장 없이 클립보드로 바로 가져갈 수 있어 편리합니다.
